PowerShell, Bash, and CMD Command Equivalents

PowerShell (aliases) Bash Command-Line Action
Get-ChildItem (ls, dir) ls dir List files
Set-Location (cd, chdir) cd cd or chdir Change directory
Get-Location (pwd) pwd cd (no arguments) Print working directory
Clear-Host (cls, clear) clear cls Clear screen
Get-Content (cat, type) cat or less type Display file contents
Copy-Item (cp, copy) cp copy or xcopy Copy file
Move-Item (mv, move, ren) mv move or ren Move/rename file
Remove-Item (rm, del, erase) rm del or erase Delete file
New-Item -ItemType Directory (mkdir, md) mkdir mkdir or md Create directory
Remove-Item (rmdir, rd) rmdir or rm -r rmdir or rd Remove directory
Get-ChildItem -Filter or dir -filter find or locate dir /s Find files
Select-String (sls) grep findstr Search in files
Get-ChildItem env: or dir env: printenv or env set Display environment variables
$env:VAR = "value" export VAR=value set VAR=value Set environment variable
Get-History (history, h) history doskey /history Display command history
Get-Help (help, man) man or --help command /? Get help
Get-NetIPConfiguration (ipconfig) ifconfig or ip a ipconfig Network information
Test-Connection (ping) ping ping Ping
Get-Process (ps) ps tasklist Process list
Stop-Process (kill) kill taskkill Kill process
