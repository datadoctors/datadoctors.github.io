PostEntry Class

The PostEntry class represents a single entry in the batch being posted to the accounting software integrated with RMS.

 

When To Use

The PostEntry class is used to perform the following tasks.

  • View information about an entry in a posted batch.

 

Procedures

The PostEntry class does not have any public procedures.

 

Properties

The PostEntry class has the following properties.

 

Amount

Data type: Currency

Access type: Read-only

Gets the amount of the post entry.

 

CreditAccount

Data type: String

Access type: Read-only

Gets the name of the credit account in the accounting software to which to post the entry.

 

DebitAccount

Data type: String

Access type: Read-only

Gets the name of the debit account in the accounting software to which to post the entry.

 

Description

Data type: String

Access type: Read-only

Gets the description of the post entry.

 

DetailID

Data type: Long

Access type: Read-only

Gets the ID of the record in the RMS database from which the information for the post entry is retrieved.

 

FieldName

Data type: String

Access type: Read-only

Gets the name of the field in the RMS database from which the information for the post entry is retrieved.

 

Key

Data type: String

Access type: Read-only

Gets the key of the post entry.

 

TableName

Data type: String

Access type: Read-only

Gets the name of the table in the RMS database from which the information for the post entry is retrieved.
