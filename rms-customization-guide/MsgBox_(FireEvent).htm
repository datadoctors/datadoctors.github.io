MsgBox Event

The handler for the MsgBox event displays a message dialog box on the POS screen.

Handler Syntax

Public Function MsgBox( _

  Prompt As Variant, _

  Buttons As VbMsgBoxStyle, _

  Title As Variant _

) As VbMsgBoxResult

Parameters

Prompt

Variant that contains the expression to display in the dialog box.

Buttons

A VbMsgBoxStyle value that specifies the buttons to display on the message box. In JScript you must supply the integer value because this enumeration is not recognized by JScript; for example, pass zero into this parameter to display an OK button on the message box.

Title

Variant that contains a string to display in the title bar of the dialog box.

Return Values

A VbMsgBoxResult value that contains the return value of the dialog box.

UI Equivalent Key

None

Remarks

The structure of the FireEvent function does not allow optional parameters. As a result, parameters that are optional in the Visual Basic MsgBox function must be explicitly passed to the FireEvent function. For example, the Buttons parameter is optional when using the Visual Basic MsgBox function; if it is not supplied, the default value VbOkOnly (0) is used. However, the handler for this event requires you to pass in a value to the Buttons parameter.

Example

The following example shows how to use the FireEvent function to display a simple message box with an OK button.

JScript

qsBridge.FireEvent("MsgBox", myVal, 0, "");
