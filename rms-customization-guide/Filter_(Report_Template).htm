Filter Report Template

The Filter defines extra or default filters applied to the basic query. It contains the following properties:

 

Property

Description

FieldName

String

Name of column to filter.

FilterOp

Filter by one of the following operations:

  • reportfilteropBetween

  • reportfilteropEqual

  • reportfilteropNotEqual

  • reportfilteropGreater

  • reportfilteropLess

  • reportfilteropGreaterEqual

  • reportfilteropLesserEqual

  • reportfilteropLike

FilterLoLim

String

Minimum value.

FilterHiLim

String

Maximum value.

The following are examples of valid values for the FilterLoLim and FilterHiLim properties:

  • "<YearStart>"

  • "<Today>"
