Column Report Template

The Column defines extra or default column properties applied to the basic query. It contains the following properties:

 

Property

Description

FieldName

String

Name of the column.

DrillDownFieldName

String

DrillDown Key, or an empty string ("") for no DrillDown. For more details, see Valid DrillDown Keys.

DrillDownReportName

String

Name of the report to launch on DrillDown. For more details, see Valid DrillDown Keys.

Title

String

Title of the report column.

The following special values may also be used (where "#" is an integer between 1 and 4):

  • !CaptionCustomerText#

  • !CaptionCustomerDate#

  • !CaptionCustomerNumber#

These titles are replaced with the corresponding custom field titles specified on the Configuration form in Store Operations, Manager.

VBDataType

Data type of the column. Controls the default data format.

  • vbString

  • vbInteger

  • vbSingle

  • vbLong

  • vbDouble

  • vbCurrency

  • vbBoolean

  • vbDate

Formula

String

SQL formula if data is calculated, or an empty string ("") if no calculation is to be used.

ColHidden

  • True

  • False

ColNotDisplayable

  • True

  • False

FilterDisabled

  • True

  • False

ColWidth

Number

Width of column in twips (1440 twips = 1 inch)

ColAlignment

  • flexAlignLeftTop

  • flexAlignLeftCenter

  • flexAlignLeftBottom

  • flexAlignCenterTop

  • flexAlignCenterCenter

  • flexAlignCenterBottom

  • flexAlignRightTop

  • flexAlignRightCenter

  • flexAlignRightBottom

  • flexAlignGeneral

GroupMethod

  • groupmethodNone

  • groupmethodSum

  • groupmethodPercent

  • groupmethodCount

  • groupmethodAverage

  • groupmethodMax

  • groupmethodMin

ColFormat

String

Standard Visual Basic format string.
