Help Files

Each program or component in Retail Management System comes with its own Help file. In addition to step-by-step instructions for using the program or component, each Help file contains reference information for the menu commands and dialog boxes you'll use. Store Operations.chm, the Help file for both Store Operations Manager and Store Operations POS, also contains a guide to the POS function keys.

Open Help within a program or component through the Help menu, or open a Help file from the Start menu by pointing to Programs, pointing to Microsoft Retail Management System, pointing to Documentation, and then clicking the Help file you want.

The OPOS Setup Help file is only accessible from within the Configure Microsoft OPOS Service Objects utility. Click Start menu, point to Programs, point to Microsoft Retail Management System, click Configure Microsoft OPOS Service Objects, and then click Help.

Tip: The Index and Search features in each Help file provide quick and easy access to the information you need.

Printed Books

The content of the Help files is also available in the Retail Management System printed books. Included in the Store Operations box are the Store Operations Getting Started Guide and Store Operations User's Guide. In the Headquarters box, you'll find the Headquarters Getting Started Guide and Headquarters User's Guide. The Getting Started guides provide information about installation and database setup, while the user guides cover daily use of Retail Management System.

QuickStart Files

After installing each Retail Management System program you will have an opportunity to view the QuickStart file for that program. (The files are also available on the Start menu in the Microsoft Retail Management System\Documentation folder.) The QuickStart files contain instructions to quickly create a database, set configuration options, and run the Manager and POS programs.

Microsoft PartnerSource Website

Join the Microsoft community! Log on at https://mbs.microsoft.com/partnersource. To view the Retail Management System home page on PartnerSource, point to Products, and then click Retail Management.

Microsoft Retail Management System Implementation Guide

The Microsoft Retail Management System Implementation Guide offers detailed implementation, installation, and configuration guidelines for Retail Management System. It covers all aspects of planning a Retail Management System implementation, including hardware and software, Store Operations and Headquarters, Microsoft SQL Server 2000, security, loss prevention, disaster recovery, accounting integration, and payment processing. It also provides step-by-step installation instructions, maintenance and change management guidelines, and troubleshooting information.

Download the Implementation Guide from PartnerSource at https://mbs.microsoft.com/partnersource/products/rms/documentation/installationsetupguides. You must have a PartnerSource or CustomerSource account to access the guide.

Training Manuals and Courses

Microsoft offers training courses for both Store Operations and Headquarters. In addition to the traditional classroom courses offered previously, online eCourses are now available. Sign up for the course format you prefer at https://mbs.microsoft.com/eventregext/default.aspx. To view the list of courses, click Partner, and then click Retail Management.

The training manuals that correspond to each of the classroom courses have been fully updated for version 1.2. These manuals cover subject matter similar to that provided in the printed books and online Help, but in lesson format. They can be ordered three ways:

  • By sending an e-mail message to mbsorder@microsoft.com. Your message should specify the training manuals (Store Operations or Headquarters) that you want to purchase.

  • By calling 1-800-456-0025 (Monday through Friday, 8 A.M. to 5 P.M. CST).

  • By using the Other Items online order form at the bottom of the Orders page at https://mbs.microsoft.com/orders/orders.aspx. On the order form page, expand Training, and then click Retail Management.
