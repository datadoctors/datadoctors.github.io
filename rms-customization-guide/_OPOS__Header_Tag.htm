OPOS Element

The OPOS element defines information for OPOS printers.

Attributes

None

Value

None

Child Elements

Element

Content

Description

PAPERCUTPERCENT

Numeric expression

Optional. Value between zero and 100 that indicates the percentage of the receipt to cut when printing is complete.

PRINTRECEIPT

Boolean expression

Optional. Determines whether to print to a receipt station.

PRINTSLIP

Boolean expression

Optional. Determines whether to print to a slip station.

PRINTJOURNAL

Boolean expression

Optional. Determines whether to print to a journal station.

SLIPWIDTH

Numeric expression

Optional. Width of the slip.

SLIPHEIGHT

Numeric expression

Optional. Height of the slip.

CHARACTERSET

  • 0 (ANSI)

  • 437 (OEM)

Optional. Indicates whether to use the ANSI or OEM code page for the OPOS printer.

Example

The following example uses the OEM character set to print receipts and journals, but not slips.

<OPOS>

    <CHARACTERSET>       437   </CHARACTERSET>

    <PAPERCUTPERCENT>    100   </PAPERCUTPERCENT>

    <PRINTRECEIPT>       True  </PRINTRECEIPT>

    <PRINTJOURNAL>       True  </PRINTJOURNAL>

    <PRINTSLIP>          False </PRINTSLIP>

</OPOS>
