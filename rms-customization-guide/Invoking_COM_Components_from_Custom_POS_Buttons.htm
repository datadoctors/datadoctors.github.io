Invoking COM Components from Custom POS Buttons

COM Component DLLs can be activated from custom POS buttons. These buttons are created in the Store Operations Manager application, and appear on the right-hand side of the POS.

To create a custom POS button that activates a COM component add-in:

    1. Create a COM Component DLL in the Store Operations add-in folder. For more information, see Creating a COM Component Add-in.

    2. Open the Store Operations Manager application. On the Database menu, point to Registers, and then click Custom POS Buttons.

    3. Create a new POS button with the following attributes.

      • Number: Enter a unique ID number; this number controls the order in which the buttons are displayed in the POS.

      • Style: Select COM Object(Session Object).

      • Caption: Enter the text that will appear on the button.

      • Command: Enter the name of your DLL and the name of the class that contains the entry point function, separated by a period. For example, if the name of your DLL is MyDLL and the name of the class that contains the Process function is MyClass, type "MyDLL.MyClass" in the Command box.

    4. Click the OK button and exit the application. The new POS button will appear the next time that the POS application is started.
