Shipping Class

The Shipping class contains shipping information for a transaction.

 

When To Use

The Shipping class is used to perform the following tasks.

  • View or modify the shipping information for a transaction.

  • Save the shipping information as an ADO Recordset object.

 

Procedures

The Shipping class does not have any public procedures.

 

Properties

The Shipping class has the following properties.

 

Address

Data type: Address

Access type: Read-only

Gets the shipping address.

 

Carrier

Data type: Carrier

Access type: Read/Write

Gets or sets the shipping carrier.

 

Charge

Data type: Currency

Access type: Read/Write

Gets or sets the total shipping charge.

 

ChargeOnOrder

Data type: Currency

Access type: Read/Write

Gets or sets the shipping charge for items on order.

 

ChargeOverride

Data type: Boolean

Access type: Read/Write

Gets or sets a Boolean value that indicates whether to override the default shipping charge for the service.

 

ChargePrevious

Data type: Currency

Access type: Read/Write

Gets or sets the shipping charge before the most recent modification.

 

ChargePurchased

Data type: Currency

Access type: Read/Write

Gets or sets the shipping charge for purchased items.

 

Notes

Data type: String

Access type: Read/Write

Gets or sets the shipping notes.

 

SaveRs

Data type: Recordset

Access type: Read-only

Gets the shipping information as an ADO Recordset object.

 

Service

Data type: Service

Access type: Read/Write

Gets or sets the shipping service.

 

ShippingLabel

Data type: StdPicture

Access type: Read/Write

Gets or sets the shipping label.

 

TrackingNumber

Data type: String

Access type: Read/Write

Gets or sets the shipping tracking number.
