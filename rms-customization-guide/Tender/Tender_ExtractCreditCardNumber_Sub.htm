Tender.ExtractCreditCardNumber Sub

The ExtractCreditCardNumber subprocedure extracts the credit card information from the specified magnetically swiped credit card data.

Public Sub ExtractCreditCardNumber( _

  RawTrack As String, _

  Track1 As String, _

  Track2 As String, _

  Number As String, _

  Expiration As String _

)

Parameters

RawTrack

String that contains the raw data contained in the magnetic track of the credit card.

Track1

String that returns the information contained on the first track of the credit card.

Track2

String that returns the information contained on the second track of the credit card.

Number

String that returns the credit card number.

Expiration

String that returns the expiration date of the credit card.
