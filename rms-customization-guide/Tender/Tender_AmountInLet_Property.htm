Tender.AmountIn Property (Let)

The Let AmountIn property sets the total amount of the tender taken in.

Public Property Let AmountIn( _

  ByVal Value As Currency _

)

Parameters

Value

Currency value that contains the total amount of the tender taken in. If this value is negative the value is made positive and transferred to the AmountOut property. The value is rounded based on the currency for the tender.

Remarks

The validation for the tender is reset if the specified value differs from the current value.
