Tender.ValidateCreditCardInfo Function

The ValidateCreditCardInfo function validates and saves the specified credit card information.

Public Function ValidateCreditCardInfo( _

  ByVal AccountHolder As String, _

  ByVal AccountNumber As String, _

  ByVal AccountExpiration As String, _

  ByVal RawNumber As String, _

  ByVal ApprovalCode As String, _

  ByVal Track1 As String, _

  ByVal Track2 As String _

) As StatusCategoryEnum

Parameters

AccountHolder

String that contains the name of the account holder for the credit card. This value is stored in the AccountHolder property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

AccountNumber

String that contains the account number for the credit card. This value is stored in the AccountNumber property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

AccountExpiration

String that contains the expiration date for the credit card. This parameter is expected to be in the MMYY format, but the "/" and "-" characters as well as empty space are acceptable and are stripped from the date. This value is stored in the AccountExpiration property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

RawNumber

String that contains the raw data from the credit card. Pass empty strings ("") to all other parameters if this information is supplied.

ApprovalCode

String that contains the approval code for the credit card. This value is stored in the ApprovalCode property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

Track1

String that contains the information contained in the first track of the credit card. This value is stored in the Track1 property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

Track2

String that contains the information contained in the second track of the credit card. This value is stored in the Track2 property. Pass an empty string ("") to this parameter if the RawNumber parameter is not an empty string ("").

Return Values

StatusCategoryEnum value that indicates whether the operation was a success.

Remarks

This function performs simple verifications on the specified information; for example, this function verifies that the AccountNumber parameter matches the mask defined in the Tender table and that the AccountExpiration parameter is in a valid date format and that the date has not already occurred.
