pricelevelEnum Enumeration

The pricelevelEnum enumerated type defines the available price levels.

Public Enum pricelevelEnum

  pricelevelDefault = 0

  priceLevelA = 1

  priceLevelB = 2

  priceLevelC = 3

End Enum

Constants

pricelevelDefault

The default price is used.

priceLevelA

The Level A price is used.

priceLevelB

The Level B price is used.

priceLevelC

The Level C price is used.

Remarks

The pricelevelEnum enumeration is used by the Transaction.SetPriceLevel function as well as by the Transaction.PriceLevel and Register.DefaultPriceLevel properties.
