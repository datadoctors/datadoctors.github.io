CurrencyDescriptor Class

The CurrencyDescriptor class contains static information about a currency.

 

When To Use

The CurrencyDescriptor class is used to perform the following tasks.

  • View unchanging descriptive information about a currency through the TenderDescriptor.CurrencyDescriptor property.

  • Convert a currency value between the local currency on the computer and this currency.

 

Procedures

The CurrencyDescriptor class defines the following procedures.

Method

Description

LocalAmount

Converts the specified amount of this currency to its equivalent in local currency.

NativeAmount

Converts the specified amount of local currency to its equivalent in this currency.

 

Properties

The CurrencyDescriptor class has the following properties.

 

Code

Data type: String

Access type: Read-only

Gets the code used to reference the currency.

 

Description

Data type: String

Access type: Read-only

Gets the description of the currency.

 

Digits

Data type: Long

Access type: Read-only

Gets the number of digits after the decimal point for the currency.

 

ExchangeRate

Data type: Double

Access type: Read-only

Gets the exchange rate between the local currency and this currency.

 

Key

Data type: String

Access type: Read-only

Gets the key of the currency.

 

LocaleID

Data type: Long

Access type: Read-only

Gets the ID of the native locale for the currency.
