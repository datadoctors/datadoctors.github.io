Register Class

The Register class represents a POS register.

 

When To Use

The Register class is used to perform the following tasks.

  • Access the OPOS devices used by the register.

  • Retrieve or set the current ZZ batch number.

  • Set the active receipt printer for the register.

 

Procedures

The Register class defines the following procedures.

Method

Description

SetActivePrinterNumber

Sets the active receipt printer for the register.

 

Properties

The Register class has the following properties.

 

ActivePrinter

Data type: ReceiptPrinter

Access type: Read-only

Gets the active receipt printer for the register.

 

Batch

Data type: Batch

Access type: Read-only

Gets the current batch for the register.

 

CashDrawer1

Data type: CashDrawer

Access type: Read-only

Gets the first cash drawer for the register.

 

CashDrawer2

Data type: CashDrawer

Access type: Read-only

Gets the second cash drawer for the register.

 

CustomButtons

Data type: QSRules.CustomButtons

Access type: Read-only

Gets the custom POS buttons for the register.

 

DefaultPriceLevel

Data type: pricelevelEnum

Access type: Read-only

Gets the default price level for the register.

 

DefaultServiceID

Data type: Long

Access type: Read-only

Gets the ID of the default shipping service for the register.

 

ID

Data type: Long

Access type: Read-only

Gets the ID of the register.

 

Keyboard

Data type: Keyboard

Access type: Read-only

Gets the touchscreen keyboard for the register.

 

Macros

Data type: QSRules.Macros

Access type: Read-only

Gets the macros for the register.

 

MICR

Data type: MICR

Access type: Read-only

Gets the Magnetic Ink Character Recognition (MICR) device used by the register to read checks.

 

MSR

Data type: MSR

Access type: Read-only

Gets the Magnetic Stripe Reader (MSR) device used by the register to swipe credit and debit cards.

 

NetDisplay

Data type: NetDisplay

Access type: Read-only

Gets the Net Display for the register.

 

Number

Data type: Long

Access type: Read-only

Gets the register number.

 

PINPad

Data type: PINPad

Access type: Read-only

Gets the Personal Identification Number (PIN) pad device for the register.

 

PoleDisplay

Data type: PoleDisplay

Access type: Read-only

Gets the pole display for the register.

 

ReceiptPrinter

Data type: ReceiptPrinter

Access type: Read-only

Gets the first receipt printer for the register.

 

ReceiptPrinter2

Data type: ReceiptPrinter

Access type: Read-only

Gets the second receipt printer for the register.

 

Scanner

Data type: Scanner

Access type: Read-only

Gets the barcode scanner for the register.

 

SigCap

Data type: SigCap

Access type: Read-only

Gets the signature capture device for the register.

 

SlipPrinter

Data type: SlipPrinter

Access type: Read-only

Gets the slip printer for the register.

 

WeightScale

Data type: WeightScale

Access type: Read-only

Gets the scale used by the register to weigh items.

 

ZZBatchNumber

Data type: Long

Access type: Read/Write

Gets or sets the number of the current ZZ batch for the register.
