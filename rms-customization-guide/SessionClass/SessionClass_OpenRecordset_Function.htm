SessionClass.OpenRecordset Function

The OpenRecordset function returns the Microsoft ActiveX Data Objects (ADO) record set retrieved by the specified SQL command.

Public Function OpenRecordset( _

  ByRef SQL As String, _

  ByRef AsReadOnly As Boolean _

) As ADODB.Recordset

Parameters

SQL

String that contains the SQL command used to determine the record set returned.

AsReadOnly

Boolean value that specifies whether the returned record set should be read-only. Pass in a value of TRUE to return a read-only record set; otherwise, pass FALSE to return a read/write record set.

Return Values

ADO Recordset object that contains the results of the SQL command. The Recordset object will be readable and writable if FALSE is passed into the AsReadOnly parameter; otherwise, the object will be read-only.

Remarks

This function provides access to any table in the Store Operations database.
