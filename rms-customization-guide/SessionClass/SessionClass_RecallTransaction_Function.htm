SessionClass.RecallTransaction Function

The RecallTransaction function recalls the specified transaction.

Public Function RecallTransaction( _

  ByVal RecallType As Transaction.recallEnum, _

  ByRef ID As Integer _

) As StatusCategoryEnum

Parameters

RecallType

A recallEnum value that specifies the type of transaction to recall.

ID

Integer value that contains the ID of the transaction to recall. The table in the database to which this ID applies depends on the value of the RecallType parameter as follows:

RecallType Value

Table.Column

recallOrder (4)

Order.ID

recallTransactionOnHold (5)

TransactionHold.ID

default

Transaction.TransactionNumber

Return Values

StatusCategoryEnum value that indicates whether the operation was a success.
