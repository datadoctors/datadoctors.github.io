Yahoo! Store Integration

With an add-in in place, Microsoft® Retail Management System Store Operations can process orders from most e-commerce services. In this section, as an example, we'll discuss creating an add-in for Yahoo! Merchant Services in XML format. Yahoo! provides an e-commerce store hosting solution that retailers can use to receive orders on the Internet. Store owners can download orders manually or have them automatically posted to a static IP address.

Retail Management System processes orders in the Store Operations database. These orders are stored in the Exchange table in the database. To begin order processing, a cashier presses CTRL+SHIFT+F10 to display the Open Internet Orders dialog box, which displays a list of the unprocessed orders in the Exchange table. The cashier selects an order and then follows a wizard to post the order as a sales transaction. The wizard also checks the XML data for errors and adds new customers and shipping addresses if found.

The Yahoo! Store Integration section contains the following topics.

  • Yahoo! Manual Integration: Describes how to manually download XML orders from Yahoo! and parse the data into the Exchange table.

  • Yahoo! Post to IP Integration: Describes how to modify the manual integration solution to automatically post the XML orders from Yahoo! to a static IP address.

  • Web Site Upload Integration:  Describes how to upload store and inventory data from Retail Management System to Yahoo!.

  • XML Format – Retail Management System Mapping: Shows how each XML element in a Yahoo! order is used by Retail Management System.

  • XML Sample of Downloaded Order: Provides an XML example of a downloaded order from Yahoo!.

For more information about Yahoo! Merchant Services, see http://smallbusiness.yahoo.com.
