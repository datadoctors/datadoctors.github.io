SetCustomer Event

The handler for the SetCustomer event sets the customer for the current transaction.

Handler Syntax

Public Function SetCustomer( _

  ByVal CustomerID As Long _

) As Boolean

Parameters

ID

Long value that contains the ID in the database of the customer to set for the current transaction.

Return Values

Boolean value that indicates whether the operation was a success. The event handler returns TRUE if the operation was a success; otherwise, FALSE.

UI Equivalent Key

F7 then select a valid customer

Remarks

The handler for the SetCustomer event performs the following actions.

  1. If the price level or discount for the current transaction has been modified from the default, the user is prompted to specify whether the discount for the new customer overrides the current transaction discount.

  2. If a customer is currently set for the transaction the customer is removed.

  3. If the specified customer is found in the database the customer information is added to the transaction.

  4. Days overdue are calculated on open account receivables.

  5. If the automatic shipping is configured and the customer has a default shipping service, the shipping service on the transaction is set to the customer’s default service.

  6. If the tax basis is customer tax, taxes and totals are recalculated.

  7. If the price levels or discounts are overridden and the cashier does not have sufficient privilege, the customer is cleared from the transaction. Otherwise, the price level and discount are set to the new customer defaults.

  8. If the customer is tax exempt, taxes are disabled.

  9. If the customer has a dedicated sales representative the sales representative for the transaction is set to the customer default.

  10. Prices and totals are recalculated and the customer purchase limits are validated.

Example

The following example shows how to use the FireEvent function to set the customer for the current transaction.

JScript

function myLoadCustomer()

{

  custID = qsBridge.FireEvent("SelectCustomer");

  qsBridge.FireEvent("SetCustomer", custID);

}
