Active Reports

Active Reports are generated from the information in the Report Template Language Definition files (.qrp files). Users are able to generate and customize a wide variety of reports from these files. They can also sort or group data directly on the report preview window.

To list a custom report on the Reports menu in Store Operations Manager and Headquarters Manager, begin the report file name with the word "Custom - ". For example, if a report file name is "Custom – My Report.qrp", it is automatically displayed as "My Report" on the Custom Reports menu.

To use multiple data sources in Active Reports, prefix the table name with the database name. The following SQL example shows how to include data from both the RMS database and a user-defined database.

RMSDatabase.dbo.Customer CU INNER JOIN MyDatabase.dbo.MyTable MT ON MT.CustomerID = CU.ID
