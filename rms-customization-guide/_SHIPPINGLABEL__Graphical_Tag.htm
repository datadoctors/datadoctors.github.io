SHIPPINGLABEL Element

The SHIPPINGLABEL element displays a shipping label on the receipt.

Attributes

None

Value

None

Child Elements

Element

Content

Description

WIDTH

Numeric expression

Required. Width of the shipping label.

HEIGHT

Numeric expression

Required. Height of the shipping label.

TOP

Numeric expression

Optional. Position of the top of the shipping label.

LEFT

Numeric expression

Optional. Position of the left side of the shipping label.

ALIGN

Alignment string

Optional. Horizontal alignment of the shipping label.
