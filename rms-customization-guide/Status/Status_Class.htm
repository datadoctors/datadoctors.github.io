Status Class

The Status class contains information about the current status.

 

When To Use

The Status class is used to perform the following tasks.

  • Determine whether an error has occurred.

  • Retrieve information about the status of an operation.

 

Procedures

The Status class does not have any public procedures recommended for external use.

 

Properties

The Status class has the following properties.

 

Category

Data type: StatusCategoryEnum

Access type: Read-only

Gets the category of the most recent status code.

 

Code

Data type: StatusCodeEnum

Access type: Read-only

Gets the most recent status code.

 

OK

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether the most recent operation was a success.

 

Parameter1

Data type: String

Access type: Read-only

Gets the first status parameter, which contains more information about the status.

 

Parameter2

Data type: String

Access type: Read-only

Gets the second status parameter, which contains more information about the status.

 

Parameter3

Data type: String

Access type: Read-only

Gets the third status parameter, which contains more information about the status.
