Cashier Class

The Cashier class represents a single cashier. Only one Cashier object at a time exists under the main SessionClass object; this Cashier object represents the cashier currently logged on to the POS.

 

When To Use

The Cashier class is used to perform the following tasks.

  • View the cashier's limits and privileges.

  • Access the messages for the cashier.

 

Procedures

The Cashier class does not have any public procedures.

 

Properties

The Cashier class has the following properties.

 

CashDrawer

Data type: CashDrawer

Access type: Read-only

Gets the default cash drawer for the cashier.

 

CashDrawerNumber

Data type: Integer

Access type: Read-only

Gets the number of the default cash drawer for the cashier.

 

EmailAddress

Data type: String

Access type: Read-only

Gets the e-mail address of the cashier.

 

FailedLogOnAttempts

Data type: Long

Access type: Read-only

Gets the number of failed logon attempts for the cashier before the current successful logon.

 

FirstName

Data type: String

Access type: Read-only

Gets the first name of the cashier.

 

FloorLimit

Data type: Currency

Access type: Read-only

Gets the largest non-cash sale amount the cashier is allowed to tender.

 

HasPrivilege

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether the cashier has the specified privilege.

 

ID

Data type: Long

Access type: Read-only

Gets the ID of the cashier.

 

LastName

Data type: String

Access type: Read-only

Gets the last name of the cashier.

 

Loaded

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether the cashier is loaded.

 

MaxOverShortAmount

Data type: Currency

Access type: Read-only

Gets the maximum over/short error amount allowed for the cashier.

 

MaxOverShortPercent

Data type: Double

Access type: Read-only

Gets the maximum over/short error percentage allowed for the cashier.

 

Messages

Data type: CashierMessages

Access type: Read-only

Gets the messages for the cashier.

 

Name

Data type: String

Access type: Read-only

Gets the full name of the cashier.

 

Number

Data type: String

Access type: Read-only

Gets the user-defined cashier number.

 

OverShortLimitType

Data type: overshortlimitEnum

Access type: Read-only

Gets the type of the over/short error limit for the cashier.

 

ReturnLimit

Data type: Currency

Access type: Read-only

Gets the largest return transaction amount that the cashier is allowed to tender.

 

SecurityLevel

Data type: Integer

Access type: Read-only

Gets the security level of the cashier.

 

UnreadMessageCount

Data type: Long

Access type: Read-only

Gets the number of messages for the cashier that have not been read.
