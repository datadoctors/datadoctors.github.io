QSBridge.FireEvent Function

The FireEvent function invokes the specified QSBridge event, with optional parameters if specified.

Public Function FireEvent( _

  EventName As String, _

  ParamArray Parameters() _

) As Variant

Parameters

EventName

Name of the event to fire. For more information about the available events, see QSBridge Events.

Parameters()

ParamArray argument that contains parameters passed to the event. This parameter is excluded if the event does not take any parameters; however, if the event takes parameters, all parameters must be specified. For an example of an event for which all parameters must be specified, see the MsgBox event.

Return Values

Variant object that contains any information returned from the fired event.

Remarks

This function resides in the user interface workflow layer.

Example

The following example uses the FireEvent function to cancel the current transaction.
