COM Component Add-ins

COM component add-ins provide a powerful, flexible way to extend Retail Management Systems to meet the special needs of your business. These add-ins are implemented as in-process ActiveX DLLs that can be triggered at specific points in the transaction flow or activated from a custom POS button.

Some complex add-ins consist of multiple procedures that are invoked from several hooks or several custom POS buttons.

Use COM components invoked from hooks to implement the following solutions. For more information about invoking COM components from hooks, see Invoking COM Components from Hooks.

  • Adding an extended discount to a transaction.

  • Performing special processing on a payment.

  • Integrating accounting software with RMS.

  • Integrating a PIN pad with RMS.

Use COM components invoked from a custom POS button to implement the following solutions. For more information about invoking COM components from custom POS buttons, see Invoking COM Components from Custom POS Buttons.

  • Extending the shipping functionality in RMS.
