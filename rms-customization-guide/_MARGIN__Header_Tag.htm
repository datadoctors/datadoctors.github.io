MARGIN Element

The MARGIN element defines the margins for the receipt template.

Attributes

None

Value

None

Child Elements

Element

Content

Description

COPIES

Numeric expression

Required. Number of copies of the receipt to print.

TOP

Numeric expression

Required. Height of the top margin.

BOTTOM

Numeric expression

Required. Height of the bottom margin.

LEFT

Numeric expression

Required. Width of the left margin.

RIGHT

Numeric expression

Required. Width of the right margin.

ORIENTATION

  • orLandscape

  • orPortrait

Required. Indicates the orientation of the receipt.

HEADER

Numeric expression

Optional. Height of the header margin.

FOOTER

Numeric expression

Optional. Height of the footer margin.

 
Remarks

The MARGIN element is required.

Example

The following example shows a typical margin setup for a receipt template.

<MARGIN>

    <COPIES>      1          </COPIES>

    <TOP>         0          </TOP>

    <BOTTOM>      0          </BOTTOM>

    <LEFT>        0          </LEFT>

    <RIGHT>       0          </RIGHT>

    <ORIENTATION> orPortrait </ORIENTATION>

</MARGIN>
