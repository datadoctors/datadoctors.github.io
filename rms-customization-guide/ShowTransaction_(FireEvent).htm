ShowTransaction Event

The handler for the ShowTransaction event displays the transaction grid in place of the HTML browser.

Handler Syntax

Public Function ShowTransaction( _

) As Boolean

Return Values

Boolean value that indicates whether the operation was a success. The event handler returns TRUE if the operation was a success; otherwise, FALSE.

UI Equivalent Key

ESC when a Web page is displayed on top of the transaction grid
