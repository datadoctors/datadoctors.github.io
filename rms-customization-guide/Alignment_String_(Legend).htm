Alignment String

The alignment string is a special code used to specify the alignment of text or tables:

Alignment

Code

Center

"^"

Left

"&lt;"

Right

"&gt;"

Truncate text

"~" (suffix)

The following example shows how to use an alignment string.

  • Left alignment with text truncation:  "&lt;~"
