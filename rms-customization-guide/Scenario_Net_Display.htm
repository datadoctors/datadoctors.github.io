Loyalty Add-in Net Display

The loyalty program Net display shows customer loyalty savings once a customer is loaded. The information displayed depends on the loyalty status of the customer as follows:

  • Customer not enrolled: Displays the Net display message defined in Manager; the default message is "Join the loyalty program and save up to 15%!" This message appears on the Net display as shown in the following diagram.

  • Customer enrolled: Displays the customer's loyalty savings for the current transaction, for example "You have saved $3.40 in the transaction." The displayed savings is the sum of the savings on all items currently in the transaction. This message appears on the Net display as shown in the following diagram.
