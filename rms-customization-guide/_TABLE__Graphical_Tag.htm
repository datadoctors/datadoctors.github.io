TABLE Element

The TABLE element displays a table on the receipt.

Attributes

None

Value

None

Child Elements

Element

Content

Description

COLUMNHEADER

COLUMNHEADER element

Required if a ROW element exists. Defines a column in the table.

ROW

ROW element

Optional. Defines a row in the table.

TOP

Numeric expression

Optional. Position of the top of the table. Use this element to explicitly set the position of a table if it follows a BARCODE or PICTURE element.

BORDER

  • tbNone

  • tbBottom

  • tbTop

  • tbTopBottom

  • tbBox

  • tbColumns

  • tbColTopBottom

  • tbAll

  • tbBoxRows

  • tbBoxColumns

Optional. Border type of the table.

PENSTYLE

  • psSolid

  • psDash

  • psDot

  • psDashDot

  • psDashDotDot

  • psTransparent

  • psInsideSolid

Optional. Pen style for all borders for the table. This element is not used if the value of the PENWIDTH element is greater than one.

PENCOLOR

Color

Optional. Color of all borders for the table. No shade is used if this element is not specified.

PENWIDTH

Numeric expression

Optional. Width of the pen used to draw the outside border of the table and the border between the header row and the rest of the rows. This element is overridden if the PENTB and PENLR elements are defined.

PENTB

TEXT element that contains a numeric expression

Optional. Width of the pen used to draw the vertical lines in the outside border of the table.

PENLR

 

TEXT element that contains a numeric expression

Optional. Width of the pen used to draw the horizontal lines in the outside border of the table.

PEN

TEXT element that contains a numeric expression

Optional. Width of the pen used to draw the borders between the content rows and the borders between the columns.

FONT

String expression

Optional. Font style of the table. This must be the name of an existing FONT element in the receipt template; specify an empty string ("") to use the default font.

HEADERSHADE

Color

Optional. Background color of the header row. No shade is used if this element is not specified.

BODYSHADE

Color

Optional. Background color of the body rows. No shade is used if this element is not specified.

EXTENDLASTROW

Boolean expression

Optional. Determines whether to extend the last row to fill the remaining space on the receipt. If TRUE, the table fills the entire remaining row on the receipt, even if there are only a few rows in the table.

EXTENDLASTROWMARGIN

Numeric expression

Optional. Height of the margin between the last entry in the table and the footer or the bottom margin of the receipt. If the space remaining on the receipt page is less than this value, a new page is created and the table is extended down to the reserved height. This element is used to reserve space for Total or Summary tables that must appear at the end of a potentially long table of entries but not in the footer.

Example

The following example displays a table that contains the description and price of each item in the transaction.

<SET name="PageWidth" custom="true" description="Width of the page" type="vbString">

    2.75

</SET>

 

<TABLE>

    <BORDER>      tbAll        </BORDER>

    <PEN> <TEXT> 20 </TEXT> </PEN>

    <PENTB> <TEXT> 50 </TEXT> </PENTB>

    <PENLR> <TEXT> 100 </TEXT> </PENLR>

    <HEADERSHADE> vbGreen </HEADERSHADE>

 

    <COLUMNHEADER>

        <TEXT>      "Item"  </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.7 </WIDTH>

    </COLUMNHEADER>

    <COLUMNHEADER>

        <TEXT> "Price" </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.3 </WIDTH>

    </COLUMNHEADER>

 

    <FOR each="Entry">

        <ROW> Entry.Description "|" Entry.Price </ROW>

    </FOR>

</TABLE>
