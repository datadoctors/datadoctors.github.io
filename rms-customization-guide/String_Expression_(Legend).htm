String Expression

A string expression is a nonvariable quoted string. A string expression must be enclosed in quotation marks ("); otherwise Retail Management System will attempt to evaluate it as a variable.

An example of a string expression:

  • "Your Number is... "

An example of a variable, including a numeric variable:

  • Store.Name

  • Item.Price

An example of a combination of the two:

  • "Thank you for shopping at " Store.Name
