Loyalty Add-in Samples

This section contains sample code that shows how the loyalty add-in uses customization to implement the loyalty management program. The following sample code is provided:

  • Loyalty Level Computation Sample: Shows how the loyalty add-in uses the Process function in a COM component add-in to compute the loyalty level of the current customer.

  • Loyalty Information Display Sample: Shows how the loyalty add-in uses an HTML script to display the loyalty information for the current customer in the status display.

  • Loyalty Receipt Sample: Shows how the loyalty add-in uses XML to print loyalty information on receipts.

  • Loyalty Report Sample: Shows how the loyalty effectiveness report uses SQL to query both the RMS database and the loyalty database to retrieve information.
