accountcolumnEnum Enumeration

The accountcolumnEnum enumerated type defines the types that indicate whether an account in the accounting software integrated with RMS is a credit, debit, or both.

Public Enum accountcolumnEnum

  accountcolumnCreditDebit = 0

  accountcolumnCredit = 1

  accountcolumnDebit = 2

End Enum

Constants

accountcolumnCreditDebit

The account is both a credit and a debit.

accountcolumnCredit

The account is a credit.

accountcolumnDebit

The account is a debit.
