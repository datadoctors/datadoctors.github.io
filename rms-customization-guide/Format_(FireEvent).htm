Format Event

The handler for the Format event invokes the Visual Basic Format function to return a string representation of the specified expression in the specified format.

Handler Syntax

Public Function Format( _

  Expression As Variant, _

  Format As Variant _

) As Variant

Parameters

Expression

Variant that contains the object to format as a string.

Format

Variant that contains the Visual Basic format string used to format the specified expression. To use the default format, specify an empty string ("").

Return Values

Variant that contains the string representation of the specified expression in the specified format.

UI Equivalent Key

None

Remarks

The structure of the FireEvent function does not allow optional parameters. As a result, parameters that are optional in the Visual Basic Format function must be explicitly passed to the FireEvent function. For example, the Format parameter is optional when using the Visual Basic Format function; if it is not supplied, the default format is used. However, the handler for this event requires you to pass in a value to the Format parameter.

Example

The following example shows how to use the FireEvent function to format a string as a long date string.

JScript

myVal = qsBridge.FireEvent("Format", "01/07/77", "Long Date");

// myVal contains the string "Friday, January 07, 1977"
