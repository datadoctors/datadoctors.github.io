Customer Information Template Variables

The following receipt template variables contain information about the current customer. Variables that reference a collection must be passed a key to the desired entry.

 

Variable

Description

Customer.AccountBalance

Balance on the customer's account.

Customer.AccountNumber

Account number of the customer's account.

Customer.AccountOpened

Date on which the customer's account was opened.

Customer.AssessFinanceCharges

Boolean value that indicates whether to assess finance charges for the customer's account.

Customer.ChargeAccount

Boolean value that indicates whether the customer has a charge account.

Customer.Company

Name of the customer's company.

Customer.CreditLimit

Credit limit for the customer's account.

Customer.CurrentDiscount

Discount percentage defined for the customer.

Customer.CustomCaptionDate1

Customer.CustomCaptionDate2

Customer.CustomCaptionDate3

Customer.CustomCaptionDate4

Customer.CustomCaptionDate5

Captions for the custom customer date fields.

Customer.CustomCaptionNumber1

Customer.CustomCaptionNumber2

Customer.CustomCaptionNumber3

Customer.CustomCaptionNumber4

Customer.CustomCaptionNumber5

Captions for the custom customer number fields.

Customer.CustomCaptionText1

Customer.CustomCaptionText2

Customer.CustomCaptionText3

Customer.CustomCaptionText4

Customer.CustomCaptionText5

Captions for the custom customer text fields.

Customer.CustomDate1

Customer.CustomDate2

Customer.CustomDate3

Customer.CustomDate4

Customer.CustomDate5

Values of the custom customer date fields.

Customer.CustomNumber1

Customer.CustomNumber2

Customer.CustomNumber3

Customer.CustomNumber4

Customer.CustomNumber5

Values of the custom customer number fields.

Customer.CustomText1

Customer.CustomText2

Customer.CustomText3

Customer.CustomText4

Customer.CustomText5

Values of the custom customer text fields.

Customer.DaysoverDue

Greatest number of days overdue for all outstanding account receivables for the customer.

Customer.EmailAddress

Customer's e-mail address.

Customer.Employee

Boolean value that indicates whether the customer is an employee.

Customer.FirstName

Customer's first name.

Customer.GlobalCustomer

Boolean value that indicates whether the customer is a global customer. This variable is only used in a Headquarters environment.

Customer.HomeAddress.City

City of the address of the customer.

Customer.HomeAddress.Company

Company of the address of the customer.

Customer.HomeAddress.Country

Country code of the address of the customer.

Customer.HomeAddress.FaxNumber

Facsimile number of the customer.

Customer.HomeAddress.Name

Full name of the customer on the customer's address.

Customer.HomeAddress.PhoneNumber

Telephone number of the customer.

Customer.HomeAddress.State

State code of the address of the customer.

Customer.HomeAddress.StreetAddress

First line of the address of the customer.

Customer.HomeAddress.StreetAddress2

Second line of the address of the customer.

Customer.HomeAddress.Zip

Postal code of the address of the customer.

Customer.HomeAddress.CityStateZip

City, state code, and postal code of the address of the customer.

Customer.HomeAddress.Zipcity

Postal code and city of the address of the customer.

Customer.ID

ID of the entry in the Customer table for the customer.

Customer.LastclosingDate

Most recent date that the customer's account was closed.

Customer.LastName

Customer's last name.

Customer.Lastvisit

Date of the customer’s last visit to the store.

Customer.Name

Full name of the customer.

Customer.Notes

Notes for the customer.

Customer.PreviousAccountBalance

Balance on the customer's account the last time it was closed.

Customer.PriceLevel

Customer's price level. This is one of the following values:

  • pricelevelDefault (0)

  • priceLevelA (1)

  • priceLevelB (2)

  • priceLevelC (3)

Customer.PurchaseLimitEnabled

Boolean value that indicates whether purchase limits apply to the customer.

Customer.PurchaseLimitStartDate

Start date for the customer's purchase limit.

Customer.PurchaseLimitTotalPoints

Number of purchase limit points allowed for the customer during the purchase limit period.

Customer.PurchaseLimitTotalPurchases

Total purchase amount allowed for the customer during the purchase limit period.

Customer.PurchaseLimitTotalUnits

Number of purchased items allowed for the customer during the purchase limit period.

Customer.ShiptoAddress.City

City of the customer's shipping address.

Customer.ShiptoAddress.Company

Company of the customer's shipping address.

Customer.ShiptoAddress.Country

Country of the customer's shipping address.

Customer.ShiptoAddress.FaxNumber

Facsimile number of the customer's shipping address.

Customer.ShiptoAddress.Name

Full name of the customer on the customer's shipping address.

Customer.ShiptoAddress.PhoneNumber

Telephone number of the customer's shipping address.

Customer.ShiptoAddress.State

State code of the customer's shipping address.

Customer.ShiptoAddress.StreetAddress

First line of the customer's shipping address.

Customer.ShiptoAddress.StreetAddress2

Second line of the customer's shipping address.

Customer.ShiptoAddress.Zip

Postal code of the customer's shipping address.

Customer.ShiptoAddress.CityStateZip

City, state code, and postal code of the customer's shipping address.

Customer.ShiptoAddress.Zipcity

Postal code and city of the customer's shipping address.

Customer.TaxExempt

Boolean value that indicates whether the customer is exempt from taxes.

Customer.TaxNumber

Customer's sales tax number.

Customer.TotalSales

Total sales to the customer.

Customer.TotalSavings

Total savings for the customer.

Customer.TotalVisits

Number of visits the customer has made to the store.

Customer.AccountReceivable.Pay

Boolean value that indicates whether a payment is being applied the specified account receivable.

Customer.AccountReceivable.OriginalDate

Date on which the specified account receivable was opened.

Customer.AccountReceivable.DueDate

Date on which the specified account receivable is due.

Customer.AccountReceivable.ID

ID of the entry in the AccountReceivable table for the specified account receivable.

Customer.AccountReceivable.TransactionNumber

Transaction number of the specified account receivable. This variable is only used when the account receivable is a charge.

Customer.AccountReceivable.OriginalAmount

Original amount for the specified account receivable.

Customer.AccountReceivable.Balance

Current balance of the specified account receivable.

Customer.AccountReceivable.PreviousBalance

Balance of the specified account receivable before the current payment was applied.

Customer.AccountReceivable.Payment

Amount of payment currently being applied to the specified account receivable.

Customer.AccountReceivable.Reference

Transaction number or account receivable ID for the specified account receivable. The number is prefixed with "DC – " for a debit or credit account receivable, "TR – "  for a transactional charge, and "FC - " for a finance charge. In a Headquarters environment, the number is also prefixed by the store ID.

Customer.AccountReceivables.TotalPayments

Total payment being applied for all account receivables for the customer.

Customer.AccountReceivables.AppliedCredits

Total existing credits being applied in this payment for the customer.
