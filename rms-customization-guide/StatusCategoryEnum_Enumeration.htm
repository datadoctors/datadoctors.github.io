StatusCategoryEnum Enumeration

The StatusCategoryEnum enumerated type contains values that indicate the general status of the most recent operation.

Public Enum StatusCategoryEnum

  statuscategoryServerFail = 100

  statuscategoryNotFound = 200

  statuscategoryValidation = 300

  statuscategoryInvalidOperation = 400

  statuscategoryInsufficientPrivilege = 500

  statuscategoryFileIOFail = 600

  statuscategoryHardwareFail = 700

  statuscategoryGeneral = 800

  statuscategoryWarning = 900

End Enum

Constants

statuscategoryServerFail

The database server was unable to complete the requested operation.

statuscategoryNotFound

The requested element could not be found in the database.

statuscategoryValidation

The element failed validation.

statuscategoryInvalidOperation

The requested operation is not valid.

statuscategoryInsufficientPrivilege

The current user does not have the security privilege required for the operation to succeed.

statuscategoryFileIOFail

The requested EDC file operation failed.

statuscategoryHardwareFail

A hardware failure occurred. This category is currently only used by weight scale devices.

statuscategoryGeneral

A general posting error occurred.

statuscategoryWarning

The operation succeeded, but the user received a warning.

Remarks

If the operation succeeds without errors or warnings the value zero is returned from procedures that use this enumeration as a return value.

The values in this enumeration indicate the general category of the status of an operation. The specific status is defined in the StatusCodeEnum enumeration.

The StatusCategoryEnum enumeration is used by the Status.Category property, and is returned by many functions to indicate whether the operation was a success.
