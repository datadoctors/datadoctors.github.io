TitleRow Report Template

The TitleRow defines the title rows for the report. It contains the following properties:

 

Property

Description

Text

String

Text to appear in the title row.

Font

String

Name of the font to use.

FontBold

  • True

  • False

FontSize

Number

Size of the font.

Color

String

Color of the font.
