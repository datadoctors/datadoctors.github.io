reasoncodeEnum Enumeration

The reasoncodeEnum enumerated type defines the types of reason code used for discounts, returns, and tax changes.

Public Enum reasoncodeEnum

  reasoncodeReturnToMain = 0

  reasoncodeReturnToOffline = 1

  reasoncodeInventoryMainManualAdjust = 2

  reasoncodeInventoryTransferMainToOffLineManual = 3

  reasoncodeCustomerDiscount = 4

  reasoncodeNoSale = 5

  reasoncodeTaxChange = 6

  reasoncodeAccountAdjustment = 7

  reasoncodeDrop = 8

  reasoncodePayout = 9

End Enum

Constants

reasoncodeReturnToMain

The reason code is for a return of an item to the main inventory.

reasoncodeReturnToOffline

The reason code is for a return of an item to the offline inventory.

reasoncodeInventoryMainManualAdjust

The reason code is for a manual adjustment to the main inventory.

reasoncodeInventoryTransferMainToOffLineManual

The reason code is for the manual transfer of an item from the main inventory to the offline inventory.

reasoncodeCustomerDiscount

The reason code is for a customer discount.

reasoncodeNoSale

The reason code is for a No Sale transaction.

reasoncodeTaxChange

The reason code is for a tax change.

reasoncodeAccountAdjustment

The reason code is for an adjustment to a customer account.

reasoncodeDrop

The reason code is for a cash drop.

reasoncodePayout

The reason code is for a cash payout.

Remarks

The reasoncodeEnum enumeration is used by the ReasonCode.ReasonCodeType property.
