Creating and Modifying a Receipt Template

Receipt templates can be created and modified in raw XML or by using the functionality included in Manager.

Receipt templates are accessed through the Receipt Formats dialog box. You can display the Receipt Formats dialog box as follows:

  1. Open Manager.

  2. On the Database menu, point to Registers and then click Receipt Formats.

 

Creating a Receipt Template using Manager

The easiest way to create a new receipt template is to use the UI included in Manager. Use Manager to create a new receipt template as follows:

  1. On the Receipt Formats dialog box, click the New button.

  2. On the Receipt Format Properties dialog box, enter a name and description for your template.

  3. Click on the browse button for the type of receipt you are creating and select the XML file to use as the basis for your receipt format.

  4. Select your receipt and click the Properties button to display the Receipt Properties dialog box.

  5. Click the Save As button and save the receipt format with a new name.

 

Creating a Receipt Template Manually

If you do not want to use the Receipt Properties dialog box, or if you need more flexibility than is included in the Manager UI, you can create a template manually. There are two ways to manually create a receipt template.

  • Create a new XML file and save it to the ReceiptTemplates folder. The default location of the ReceiptTemplates folder is C:\Program Files\Microsoft Retail Management System\Store Operations\ReceiptTemplates.

  • Make a copy of one of the XML files in the ReceiptTemplates folder and rename it.

 

After you have completed one of the steps above, do the following to add your receipt template to RMS.

  1. On the Receipt Formats dialog box, click the New button.

  2. On the Receipt Format Properties dialog box, enter a name and description for your new receipt template.

  3. Click on the browse button for the type of receipt you are creating and select your new XML file.

  4. Click the OK button.

 

Modifying a Receipt Template

There are two ways to modify a receipt template.

  • Use Manager as follows:

    1. Select the receipt template to modify and click the Properties button to display the Receipt Properties dialog box.

    2. Modify the attributes of the receipt template.

    3. Click the Save As button to save your changes.

  • Open the XML file for the receipt format and modify the XML directly. For more information, see Receipt Template Language. You must reselect the XML file in the Receipt Format Properties dialog box for the changes to be recognized by RMS.
