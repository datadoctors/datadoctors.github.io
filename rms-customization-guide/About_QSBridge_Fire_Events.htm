QSBridge Events

HTML script add-ins raise QSBridge events to perform operations in the Store Operations user interface workflow layer. These events are raised by calling the FireEvent function.

QSBridge defines the following events.

Event

Description

CancelTransaction

Cancels the current transaction and starts a new transaction.

ClearCustomer

Clears the current customer from the current transaction.

ClearStatus

Obsolete; do not use.

ConcludeTransaction

Tenders and posts the current transaction.

CreateBackOrder

Converts the current transaction into a back order transaction.

CreateHold

Puts the current sales transaction on hold.

CreateLayaway

Converts the current transaction into a layaway transaction.

CreatePostVoid

Voids the current sales transaction.

CreateQuote

Converts the current transaction into a quote transaction.

CreateSale

Converts the current transaction into a sales transaction.

CreateWorkOrder

Converts the current transaction into a work order transaction.

CurrencyToString

Returns a string representation of the specified currency value.

DateTimeToString

Returns a string representation of the specified date and time.

DateToString

Returns a string representation of the specified date.

DoubleToString

Returns a string representation of the specified double value.

ExecuteProgram

Launches an external Windows program and immediately returns control to Retail Management System Store Operations.

ExecuteProgramAndWait

Launches an external Windows program and waits until it completes before returning to Retail Management System Store Operations.

Format

 

Invokes the Visual Basic Format function to return a string representation of the specified expression in the specified format.

GetStatusText

 

Returns the human-readable text of the current QSRules error status.

HideCustom

Hides the custom POS buttons at the right of the POS screen.

HideFunction

Hides the function buttons at the bottom of the POS screen.

HideMe

Hides the POS.

HideStatus

Hides the status bar at the top of the POS screen.

HideTotals

Hides the Totals pane at the bottom of the POS screen.

HideWebStatus

Hides the HTML status bar at the top of the POS screen.

InitializeTransaction

Initializes a new transaction.

Login

Displays the Login dialog box and reinitializes the transaction.

Logout

Logs out of the current QSRules session.

MsgBox

Displays a message dialog box on the POS screen.

PerformAddItem

Adds an new item to the POS screen and gathers any required data, such as a serial number, matrix information, and error messages.

PerformRecall

Displays the Recall dialog box.

PerformTransactionCalculate

Displays the calculator.

PerformTransactionLookup

Displays a Lookup dialog box depending on the column in the transaction grid that has the focus.

PostDrawerFunction

Displays the Drawer Function dialog box and posts the transaction.

PostDrop

Posts a cash drop transaction.

PostNoSale

Posts a No Sale transaction.

PostOpenClose

Displays the Open/Close dialog box and posts the transaction.

PostPayment

Displays the Receive Payment dialog box and posts a payment to the selected Account Receivable.

PostPayout

Posts a payout transaction.

PostTransaction

Posts the current transaction and displays the Posting Status dialog box.

PostTransactionUnattended

Posts the current transaction without displaying the Posting Status dialog box.

PrintLastReceipt

Reprints the most recent receipt.

PrintXReport

Prints and posts the X report.

PrintZReport

Prints and posts the Z report.

PrintZZReport

Prints and posts the ZZ report.

Quit

Closes the POS screen.

RecallBackOrder

Displays the Back Orders dialog box.

RecallHold

Displays the Transactions on Hold dialog box.

RecallLayaway

Displays the Layaways dialog box.

RecallQuote

Displays the Quotes dialog box.

RecallWorkOrder

Displays the Work Orders dialog box.

RefreshDisplay

Refreshes the POS transaction screen and pole display.

Secure

Logs the current cashier out of the POS and displays the Login dialog box.

SelectBackOrder

Displays the Back Orders dialog box and returns the ID of the selected open back order.

SelectCustomer

Displays the Customers dialog box and returns the ID of the selected customer.

SelectHoldTransaction

Displays the Transactions On Hold dialog box and returns the ID of the selected held transaction.

SelectItem

Displays the Items dialog box and returns the item lookup code of the selected item.

SelectLayaway

Displays the Layaways dialog box and returns the ID of the selected layaway.

SelectProcess

Displays the Select a Process to Activate dialog box and activates the selected running process.

SelectQuote

Displays the Quotes dialog box and returns the ID of the selected quote.

SelectSalesRep

Displays the Sales Reps dialog box and returns the ID of the selected sales representative.

SelectShippingEdit

Displays the Pending Shipping Records dialog box and returns the ID of the selected shipping record.

SelectSubstituteItem

Displays the Substitute Item dialog box for the specified item and returns the ID of the selected substitute item.

SelectTransaction

Displays the Transactions dialog and returns the ID of the selected transaction.

SelectWorkorder

Displays the Work Orders dialog box and returns the ID of the selected work order.

SetCustomer

Sets the customer for the current transaction.

SetGridDefaultsEdit

Sets the specified grid to the look and feel used by the editable grids in the POS and Manager programs.

SetGridDefaultsList

Sets the specified grid to the look and feel used by the lists in the POS and Manager programs.

SetStatus

Sets the text to be displayed in the status window in the lower left corner of the POS screen.

SetSubstituteItem

Replaces the current transaction entry with the specified item.

SetTransactionSalesRep

Sets the sales representative for the transaction.

ShowAbout

Displays the About Store Operations dialog box.

ShowAutoGenerate

Displays the Auto-Generate Sales dialog box.

ShowCalendar

Displays the Calendar dialog box.

ShowCalendarEventReminder

Displays the Calendar Event Reminder dialog box.

ShowClosingAmounts

Displays the Closing Amounts dialog box, which allows the cashier to enter the closing amounts for the session.

ShowCustom

Displays the custom POS buttons at the right of the POS screen.

ShowDiscount

Displays the Set Discount/Price Level dialog box.

ShowDisplayProperties

Displays the Display Properties dialog box.

ShowFunction

Displays the function keys at the bottom of the POS screen.

ShowGasPumpDesposits

Displays the Gas Pump Deposit dialog box.

ShowGraphs

Displays the Report dialog box.

ShowHTML

Displays an HTML Web page in place of the transaction grid.

ShowItemComment

Displays the Comment dialog box for the current transaction entry.

ShowJournalViewer

Displays the Journal Viewer dialog box.

ShowMenu

Displays a menu selection and returns the choice made by the cashier.

ShowMessages

Displays the Messages dialog box.

ShowNewMessage

Displays the Message dialog box, which allows the cashier to create and send messages to other cashiers.

ShowOpeningAmounts

Displays the Opening Amounts dialog box, which allows the cashier to enter the opening amounts for the session.

ShowPickupEntireOrder

Picks up all remaining items on an order and tenders the transaction.

ShowPickupOrder

Displays the Pick Up Order dialog box, which allows the cashier to process a partial pickup of an order.

ShowSalesTaxSelector

Displays the Sales Tax Selector dialog box, which allows the cashier to select which taxes to apply to the transaction.

ShowSerialNumber

Displays a dialog box that allows the cashier to select the serial number for the current transaction entry.

ShowStatus

Displays the status bar at the top of the POS screen.

ShowTimeClock

Displays the Time Clock dialog box.

ShowTotals

Displays the Totals pane at the bottom of the POS screen.

ShowTransaction

Displays the transaction screen in place of the HTML Browser pane.

ShowWebStatus

Displays the HTML Status pane at the top of the POS screen.

Tender

Tenders and posts the current transaction.

TimeToString

Returns a string representation of the specified time.

ToggleReceiptPrintersOn

Toggles the receipt printers on or off.

ToggleReturnMode

Toggles the transaction between Return mode and Sales mode.

ToggleTaxableItem

Toggles the taxable status of the current transaction entry.

ToggleTaxableTransaction

Toggles the taxable status for the entire transaction.
