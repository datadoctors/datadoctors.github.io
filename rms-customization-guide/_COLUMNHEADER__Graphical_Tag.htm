COLUMNHEADER Element

The COLUMNHEADER element defines a column in a table.

Attributes

None

Value

None

Child Elements

Element

Content

Description

ALIGNMENT

Alignment string

Optional. Alignment of the text in the column.

WIDTH

Numeric expression

Optional. Width of the column.

TEXT

String expression

Optional. Text in the header row of the column. A column header can have any number of child TEXT elements.

Remarks

A table always has a header row, even if you define the columns with empty COLUMNHEADER elements.

Example

The following example displays a table that contains the description and price of each item in the transaction.

<TABLE>

    <BORDER>      tbAll        </BORDER>

    <PEN> <TEXT> 20 </TEXT> </PEN>

    <PENTB> <TEXT> 50 </TEXT> </PENTB>

    <PENLR> <TEXT> 100 </TEXT> </PENLR>

    <HEADERSHADE> vbGreen </HEADERSHADE>

 

    <COLUMNHEADER>

        <TEXT>      "Item"  </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.7 </WIDTH>

    </COLUMNHEADER>

    <COLUMNHEADER>

        <TEXT> "Price" </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.3 </WIDTH>

    </COLUMNHEADER>

 

    <FOR each="Entry">

        <ROW> Entry.Description "|" Entry.Price </ROW>

    </FOR>

</TABLE>
