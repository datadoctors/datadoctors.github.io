Invoking an HTML Add-in

Each embedded HTML browser is set up in a different way. The HTML script that will be run for each embedded HTML browser is specified as follows:

  • HTML Status Bar

  1. Open the Store Operations Administrator application and connect to the database.

  2. On the File menu, click Configuration, and then click the Register tab.

  3. Type the path of the HTML script file to use for the status bar in the Web status text box, or click the browse button to browse to the HTML script file you want to display.

  4. Click the OK button and exit the Administrator application.

 

  • HTML Browser

  1. Open the Store Operations Manager application. On the Database menu, point to Registers, and then click Custom POS Buttons.

  2. Create a new POS button with the following attributes.

    • Number: Enter a unique ID number; this number controls the order in which the buttons are displayed in the POS.

    • Style: Select HTML Window.

    • Caption: Enter the text that will appear on the button.

    • Command: Enter the filename of the HTML script to display over the transaction grid when the POS button is clicked.

  3. Click the OK button and exit the application. The new POS button will appear the next time that the POS application is started.

 

  • Net Display receipt pane

  1. Open the Store Operations POS application.

  2. Press CTRL+F6 to display the Display Properties dialog, and then click the Net Display tab.

  3. Select the Show Net Display and Show receipt on Net Display check boxes.

  4. Enter the window positions.

  5. Click the browse button to browse to the HTML script file you want to display in the receipt pane.

  6. Click the OK button.

 

  • Net Display advertising pane

  1. Open the Store Operations Manager application. On the Database menu, point to Registers, and then click Net Display Channels.

  2. Open a channel.

      • If you want to use an existing channel, highlight the channel in the selection window and then click the Properties button.

      • If you want to create a new channel or a channel does not exist, click the New button to create a new channel. Type the name of the new channel in the Channel name text box.

    2. Click the OK button and close the manager application.

  2. Press CTRL+F6 to display the Display Properties dialog, and then click the Net Display tab. From Store Operations POS, press CTRL + F6 and select the Net Display tab.

  3. Select the Show Net Display check box.

  4. Click the OK button.

 

  • Internal Browser

  1. Open the Store Operations Manager application. On the Database menu, point to Registers, and then click Custom POS Buttons.

  2. Create a new POS button with the following attributes.

    • Number: Enter a unique ID number; this number controls the order in which the buttons are displayed in the POS.

    • Style: Select Internal HTML Window.

    • Caption: Enter the text that will appear on the button.

    • Command: Enter the filename of the HTML script to display in the separate browser window when the POS button is clicked.

  3. Click the OK button and exit the application. The new POS button will appear the next time that the POS application is started.
