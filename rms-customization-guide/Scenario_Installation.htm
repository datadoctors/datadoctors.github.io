Loyalty Add-in Installation

Most elements of the loyalty add-in are set up automatically by the installation wizard. Creating a wizard to automate tasks makes installation of add-ins much easier for the users, and decreases deployment time. Wizards can automate the following tasks:

  • Creating databases

  • Putting HTML and template files in the RMS folders

  • Adding hook information to the registry

However, the following tasks must be performed in Manager or POS, and cannot be automated:

  • Setting the default receipt template

  • Modifying the Net display
