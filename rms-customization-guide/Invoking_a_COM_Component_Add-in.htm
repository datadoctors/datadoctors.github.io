Invoking a COM Component Add-in

COM component add-ins can be invoked either by a custom POS button or by a hook. However, before you deploy your COM component to the POS stations you should do the following:

  1. Copy your DLL to each sales register that will run the add-in.

  2. Register the add-in DLL on each sales register, for example by using the regsvr32 command.

  3. Set up the add-in to be invoked from a hook (see Invoking COM Components from Hooks) or a custom POS button (see Invoking COM Components from Custom POS Buttons).
