Welcome to the Microsoft Retail Management System Customization Guide

Microsoft® Retail Management System is a highly customizable product. Customization can address the needs of an individual retailer, a vertical market segment, or provide broad solutions that apply across the retail industry.

This customization guide provides a wide range of information you can use to customize Retail Management System for a variety of solutions. It includes the following sections:

  • Customization Guide Overview: Describes Retail Management System and customization.

  • Custom Add-ins: Describes how to extend the product using COM component, standard EXE, and HTML script add-ins.

  • Receipts: Describes how to create custom graphical receipts.

  • Reports: Describes how to customize Active Reports and Crystal Reports.

  • Yahoo! Store Integration: Describes how to interface with Yahoo! merchant Web sites.

  • Databases: Describes the Retail Management System databases.

  • Tailspin Toys Scenario: Contains information about the loyalty program scenario.

  • Reference: Contains detailed reference information on using Retail Management System’s business logic.
