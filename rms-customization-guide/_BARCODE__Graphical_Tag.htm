BARCODE Element

The BARCODE element displays a barcode on the receipt.

Attributes

None

Value

None

Child Elements

Element

Content

Description

TEXT

Numeric expression

Required. Value used to generate the barcode.

WIDTH

Numeric expression

Required. Width of the barcode.

HEIGHT

Numeric expression

Required. Height of the barcode.

STYLE

  • MSSNONE

  • MSS2OF5

  • MSSINTERLEAVED2OF5

  • MSS3OF9

  • MSSCODABAR

  • MSS3OF9X

  • MSSCODE128A

  • MSSCODE128B

  • MSSCODE128C

  • MSSUPCA

  • MSSMSI

  • MSSCODE93

  • MSSEXTENDEDCODE93

  • MSSEAN13

  • MSSEAN8

  • MSSPOSTNET

  • MSSANSI3OF9

  • MSSANSI3OF9X

  • MSSCODE128AUTO

  • MSSUCCEAN128

  • MSSUPCE

Optional. Style of the barcode.

UPCNOTCHES

  • msUPCNone

  • msUPCNAbove

  • msUPCNBelow

  • msUPCNBoth

Optional. Location of the UPC notches in the barcode.

TOP

Numeric expression

Optional. Position of the top of the barcode.

LEFT

Numeric expression

Optional. Position of the left side of the barcode.

 

ALIGN

Alignment string

Optional. Horizontal alignment of the barcode.

PRINTCAPTION

Boolean expression

Optional. Determines whether to print the numeric value of the barcode below the barcode.

BARWIDTH

Numeric expression

Optional. Width of the bars in the barcode.

DIRECTION

  • msDLeftToRight

  • msDRightToLeft

  • msDTopToBottom

  • msDBottomToTop

Optional. Direction to print the barcode.

FONT

String expression

Optional. Font to use for the caption of the barcode. This must be the name of an existing FONT element in the receipt template; specify an empty string ("") to use the default font.

Example

The following example shows how to display a barcode on a receipt.

<SET name="PageWidth" custom="true" description="Width of the page" type="vbString">

    2.75

</SET>

 

<BARCODE>

    <STYLE>        MSS3OF9                              </STYLE>

    <UPCNOTCHES>   msUPCNNone                           </UPCNOTCHES>

    <LEFT>         PageWidth * .30                      </LEFT>

    <WIDTH>        PageWidth * .40                      </WIDTH>

    <HEIGHT>       4                                    </HEIGHT>

    <PRINTCAPTION> True                                 </PRINTCAPTION>

    <BARWIDTH>     0                                    </BARWIDTH>

    <TEXT>         Transaction.ReceiptTransactionNumber </TEXT>

</BARCODE>
