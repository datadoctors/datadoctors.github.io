recallEnum Enumeration

The recallEnum enumerated type defines the types of transaction recall.

Public Enum recallEnum

   recallNone = 0

   RecallTransactionForReturn = 1

   recallTransactionForRepost = 2

   recallTransactionForVoid = 3

   recallOrder = 4

   recallTransactionOnHold = 5

End Enum

Constants

recallNone

The transaction was not recalled.

RecallTransactionForReturn

The transaction was recalled to return an item.

recallTransactionForRepost

The transaction was recalled to be reposted.

recallTransactionForVoid

The transaction was recalled to be voided.

recallOrder

The transaction is a recalled order.

recallTransactionOnHold

The transaction was recalled after being put on hold.

Remarks

The recallEnum enumeration is used by the SessionClass.RecallTransaction function and by the Transaction.RecallType property.
