FONT Element

The FONT element defines fonts available in the template.

Attributes

Attribute

Value

Description

name

String

Required. String used to identify the font elsewhere in the receipt template. Specify an empty string ("") to make this font the default font.

Value

None

Child Elements

Element

Content

Description

NAME

String expression

Optional. Name of the actual font to use, for example, "Arial".

SIZE

Numeric expression

Optional. Point size of the font to use.

BOLD

Boolean expression

Optional. Determines whether the font is bold.

UNDERLINE

Boolean expression

Optional. Determines whether the font is underlined.

ITALIC

Boolean expression

Optional. Determines whether the font is italicized.

COLOR

Color

Optional. Color of the font.

CPL

Numeric expression

Optional. Number of characters per line for the font.

Remarks

The FONT element is required. You must specify at least a default font by defining a FONT element with an empty string ("") for the name attribute.

Example

The following example sets the default font for the receipt template to the underlined red Arial font with a point size of eight.

<FONT name="">

    <NAME>      "Arial" </NAME>

    <SIZE>      8       </SIZE>

    <BOLD>      False   </BOLD>

    <UNDERLINE> True    </UNDERLINE>

    <ITALIC>    False   </ITALIC>

    <COLOR>     vbRed   </COLOR>

    <CPL>       40      </CPL>

</FONT>
