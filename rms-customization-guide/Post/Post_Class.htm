Post Class

The Post class contains information about the posting of a batch to the integrated accounting software.

 

When To Use

The Post class is used to perform the following tasks.

  • Retrieve information about the posting of a batch to the integrated accounting software.

  • Change the accounting file to which the batch is posted.

 

Procedures

The Post class does not have any public procedures recommended for external use.

 

Properties

The Post class has the following properties.

 

Accounts

Data type: PostAccounts

Access type: Read-only

Gets the accounts to which to post the batch.

 

BatchNumber

Data type: Long

Access type: Read-only

Gets the batch number of the batch to close.

 

ClosingTime

Data type: Date

Access type: Read-only

Gets the time at which the posted batch was closed.

 

Entries

Data type: PostEntries

Access type: Read-only

Gets the entries in the post.

 

Filename

Data type: String

Access type: Read/Write

Gets or sets the name of the accounting file to which to post.

 

RegisterNumber

Data type: Long

Access type: Read-only

Gets the register number of the register that generated the posted batch.

 

StoreID

Data type: Long

Access type: Read-only

Gets the ID of the store that generated the posted batch.

 

VatDetailID

Data type: String

Access type: Read-only

Gets the VAT detail ID used for the post.

 

VATEnabled

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether the post uses a VAT tax system.
