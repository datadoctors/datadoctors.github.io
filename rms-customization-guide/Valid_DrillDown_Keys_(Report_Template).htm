Valid DrillDown Keys for Report Templates

Retail Management Systems uses the supplied DrillDown keys in conjunction with the data in the column to automatically open the appropriate editor for that key.

For example, if the key Item.ItemLookupCode is used and the user double-clicks on that column, Retail Management Systems opens the properties screen for an Item with the matching ItemLookupCode.

Valid DrillDown keys include the following:

  • AccountReceivable.ID

  • Item.ItemLookupCode

  • Batch.BatchNumber

  • Cashier.Name

  • Category.Name

  • Customer.AccountNumber

  • Department.Name

  • Item.ItemLookupCode

  • ItemMessage.Title

  • ItemTax.Description

  • OldJournal.BatchNumber

  • Schedule.Description

  • PurchaseOrder.PONumber

  • ReasonCode.Code

  • SalesRep.Name

  • Supplier.SupplierName

  • Tax.Description

  • Tender.Description

  • [Transaction].TransactionNumber

  • Journal.ID
