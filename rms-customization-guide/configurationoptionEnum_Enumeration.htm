Constants

configurationoptionAllowNewCustomer

Option to allow the entry of new customers at the POS.

configurationoptionCalculateKitCost

Option to automatically calculate the cost of a kit from the cost of its component items.

configurationoptionDefaultGlobalCustomer

Option to create new customers as global customers by default.

configurationoptionDisplayOutOfStock

Option to display a warning message when the cashier attempts to sell more of an item than is currently in stock.

configurationoptionEnableBackOrder

Option to allow back orders of an item that is out of stock.

configurationoptionEnableGlobalCustomers

Option to configure the system to use global customers in a multistore environment.

configurationoptionEnforceUniqueSerialNumbers

Option to require the cashier to select a unique serial number for each serialized item.

configurationoptionForceLogonEachSale

Option to require the cashier to log on to the POS for each transaction.

configurationoptionForceOpenCloseAmounts

Option to require the cashier to enter opening and closing amounts before generating a Z report.

configurationoptionLookupUnknownItems

Option to display the Items dialog box if the lookup code of an item added to the transaction is not found in the database.

configurationoptionPrintNoSaleReceipt

Option to print receipts for No Sale transactions.

configurationoptionPromptForCustomer

Option to prompt the cashier to select a customer for every transaction.

configurationoptionPromptForSalesRep

Option to prompt the cashier to select a sales representative for every transaction.

configurationoptionPOUpdateSupplierCost

Option to automatically update supplier costs and tax rates when an item from a supplier is received.

configurationoptionPromptForDetails

Option to display the Transaction Information dialog box for every transaction.

configurationoptionRecordComments

Option to save comments entered for an item in the transaction history.

configurationoptionRequireCustomer

Option to require the cashier to select a customer before tendering a sales transaction.

configurationoptionRequireCustomerForOrders

Option to require the cashier to select a customer before tendering a order transaction.

configurationoptionRequireCustomerForQuotes

Option to require the cashier to select a customer before tendering a quote transaction.

configurationoptionRequireDecimalEntry

Option to require the cashier to enter a decimal point in all numbers to indicate a fraction of the currency.

configurationoptionRequireSerialNumberAtPOS

Option to require the cashier to enter a serial number for all serialized items added to a transaction.

configurationoptionReturnLocalCurrency

Option to display change in the local currency, regardless of the tender in which the transaction was tendered.

configurationoptionDisplayUnknownItemScreen

Option to display a warning when the cashier adds an unrecognized item to a transaction.

configurationoptionEnforceOutOfStock

Option to prevent the cashier from selling more of an item than is currently in stock.

configurationoptionSuppressBackupMessage

Option to prevent the Manager application from displaying the database backup reminder when the application is closed.

configurationoptionUseChangeCalculator

Option to display a change calculator when a transaction is tendered in a currency other than the local currency.

configurationoptionPOSSerialNumberCreation

Option to allow creation of serial numbers at the POS.

configurationoptionRandomWeightUPC

Option to enable price-embedded barcodes.

configurationoptionAutomaticShipping

Option to automatically add shipping costs for the default carrier and service to every transaction.

configurationoptionRequireSalesRep

Option to require the cashier to select a sales representative before tendering a transaction.

configurationoptionSynchQuantityWithSNs

Option to synchronize the quantity of serialized and voucher items with the number of serial numbers entered for the item.

configurationoptionIncludeDuplicate

Option to mark receipts printed from a journal as a duplicate receipt.

configurationoptionReasonCodePromptOnManualAdjust

Option to prompt the user to enter a reason code for manual inventory adjustment.

configurationoptionNoItemList

Option to always display the Find dialog box for items instead of the standard list.

configurationoptionNoCustomerList

Option to always display the Find dialog box for customers instead of the standard list.

configurationoptionReasonCodeRequiredOnReturns

Option to require a reason code for returns.

configurationoptionReasonCodeRequiredOnDiscounts

Option to require a reason code for discounts.

configurationoptionReasonCodeRequiredOnManualAdjust

Option to require a reason code for manual inventory adjustment.

configurationoptionReasonCodePromptOnReturns

Option to prompt the cashier to enter a reason code for returns.

configurationoptionReasonCodePromptOnDiscounts

Option to prompt the cashier to enter a reason code for discounts.

configurationoptionReasonCodeRequiredOnNoSales

Option to require a reason code for No Sale transactions.

configurationoptionReasonCodePromptOnNoSales

Option to prompt the cashier to enter a reason code for No Sale transactions.

configurationoptionReasonCodePromptOnTaxChange

Option to prompt the cashier to enter a reason code for tax changes.

configurationoptionReasonCodeRequiredOnTaxChange

Option to require a reason code for tax changes.

configurationoptionReasonCodePromptOnAccountAdjustment

Option to prompt the user to enter a reason code for adjustments to a customer account.

configurationoptionReasonCodeRequiredOnAccountAdjustment

Option to require a reason code for adjustments to a customer account.

configurationoptionPOSShowSubDescriptions

Option to show item subdescriptions in transaction entries.

configurationoptionPOSShowExtendedDescription

Option to show extended descriptions for items in transaction entries.

configurationoptionPOSShowItemNotes

Option to show notes for items in transaction entries.

configurationoptionSharePrinter

Option to share OPOS printers with other applications.

configurationoptionEnableQSNetService

Option to allow the POS to start the RMS network service.

configurationoptionRequireFullDepositOnBackOrders

Option to require the full price of an item as a deposit for back order transactions.

configurationoptionRandomWeightEAN

Option to use random weight EAN codes.

configurationoptionShowAlternateTextOnPoleDisplay

Option to display the location of the item bin on the pole display instead of the item description.

configurationoptionRequireSerialNumberOnInventoryReceive

Option to require the user to enter serial numbers for serialized items when inventory is received.

configurationoptionDropDownItemSubDescriptions

Option to display a drop-down list of item subdescriptions from which the cashier selects the subdescription to display in the transaction entry.

configurationoptionTenderQuotes

Option to save and recall tender information for quotes.

configurationoptionReasonCodeRequiredOnPhysicalInventoryCountAdjustment

Option to require a reason code for adjustments to item quantities from physical inventory counts.

configurationoptionDontRoundFoodStamps

Option to prevent rounding food stamps.

configurationoptionMaskCreditCardNumbers

Option to mask all but the last four digits of credit card numbers with the "X" character in the database.

configurationoptionReasonCodeRequiredOnDrops

Option to require a reason code for cash drops.

configurationoptionReasonCodeRequiredOnPayouts

Option to require a reason code for payouts.

configurationoptionDontRememberListSorts

Option to always sort lists by first column ascending order instead of by the last sort.

configurationoptionAllowEditClosedPuchaseOrders

Option to allow closed purchase orders to be edited.

configurationoptionConnectHQForCustomerOpenARs

Option to connect to Headquarters to retrieve open account receivables for all stores when receiving payment for a customer's account receivables.

Remarks

The configurationoptionEnum enumeration is used by the Configuration.OptionIsSet property.
