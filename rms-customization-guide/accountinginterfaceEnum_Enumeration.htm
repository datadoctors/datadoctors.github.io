accountinginterfaceEnum Enumeration

The accountinginterfaceEnum enumerated type defines the types of accounting software that can be integrated with RMS.

Public Enum accountinginterfaceEnum

  accountinginterfaceNone = 0

  accountingInterfaceMSGPSBM = 1

  accountingInterfacePeachTree20 = 2

  accountingInterfacePeachTree30 = 3

  accountingInterfaceQuickBooksFile = 4

  accountingInterfaceMYOB = 5

  accountingInterfaceBLACKBAUD6 = 6

  accountingInterfaceBLACKBAUD7 = 7

  accountinginterfaceQuickBooks2003 = 8

  accountinginterfaceRMSXML = 9

  accountingInterfaceCustomHook = 100

End Enum

Constants

accountinginterfaceNone

 No accounting software is integrated with RMS.

accountingInterfaceMSGPSBM

The accounting software integrated with RMS is Microsoft Great Plains SBM.

accountingInterfacePeachTree20

The accounting software integrated with RMS is Peachtree 2.0.

accountingInterfacePeachTree30

The accounting software integrated with RMS is Peachtree 3.0.

accountingInterfaceQuickBooksFile

The accounting software integrated with RMS is an Intuit QuickBooks file.

accountingInterfaceMYOB

The accounting software integrated with RMS is MYOB software.

accountingInterfaceBLACKBAUD6

The accounting software integrated with RMS is Blackbaud 6.0.

accountingInterfaceBLACKBAUD7

The accounting software integrated with RMS is Blackbaud 7.0.

accountinginterfaceQuickBooks2003

The accounting software integrated with RMS is Intuit QuickBooks 2003.

accountinginterfaceRMSXML

The accounting software integrated with RMS is an XML file created by RMS.

accountingInterfaceCustomHook

Accounting is handled by a custom hook within the POS. For more information, see the ExportAccounting hook.

Remarks

The accountinginterfaceEnum enumeration is used by the Configuration.AccountingInterface property.
