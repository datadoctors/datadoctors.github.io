Configuration Class

The Configuration class provides access to the configuration set in RMS Manager.

 

When To Use

The Configuration class is used to perform the following tasks.

  • Access the Manager configuration information.

 

Procedures

The Configuration class does not have any public procedures recommended for external use.

 

Properties

The Configuration class has the following properties.

 

AccountDefaultID

Data type: Long

Access type: Read/Write

Gets the ID of the default account type for new customers.

 

AccountingCompany

Data type: String

Access type: Read/Write

Gets the company used for QuickBooks integration.

 

AccountingFileName

Data type: String

Access type: Read/Write

Gets the filename of the accounting file used for QuickBooks integration.

 

AccountingInterface

Data type: accountinginterfaceEnum

Access type: Read/Write

Gets the accounting software integrated with RMS.

 

AccountMonthlyClosingDay

Data type: Integer

Access type: Read/Write

Gets the day of the month on which the billing cycle is closed for all accounts.

 

BackOrderExpirationDays

Data type: Integer

Access type: Read/Write

Gets the number of days a back order exists in RMS before it expires.

 

BillPostingAccount

Data type: String

Access type: Read/Write

Gets the bill posting account used for QuickBooks integration.

 

CaptionCustomerCustomDate

Data type: String

Access type: Read/Write

Gets the specified custom date caption.

 

CaptionCustomerCustomNumber

Data type: String

Access type: Read/Write

Gets the specified custom number caption.

 

CaptionCustomerCustomText

Data type: String

Access type: Read/Write

Gets the specified custom text caption.

 

CostUpdateMethod

Data type: costupdatemethodEnum

Access type: Read/Write

Gets the method used to update costs when inventory is received.

 

DefaultTenderID

Data type: Long

Access type: Read/Write

Gets the ID of the default tender used to return change.

 

DisplayOutOfStock

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether a warning message will be displayed when the cashier attempts to sell more than the quantity of an item in stock.

 

DisplayUnknownItemScreen

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether a warning message will be displayed when the cashier adds an unrecognized item to the transaction.

Data type: Integer

Access type: Read/Write

Gets the number of seconds before an Electronic Draft Capture (EDC) attempt times out.

 

EnableBackOrder

Data type: Boolean

EnforceOutOfStock

Data type: Boolean

Access type: Read-only

Gets a Boolean value that indicates whether the cashier is allowed to sell more than the quantity of an item in stock.

 

EnforceUniqueSerialNumbers

Data type: Boolean

ForceLogonEachSale

ForceOpenCloseAmounts
