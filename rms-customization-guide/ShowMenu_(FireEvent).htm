ShowMenu Event

The handler for the ShowMenu event displays a menu selection and returns the choice made by the cashier.

Handler Syntax

Public Function ShowMenu( _

  ByVal Caption As String, _

  ParamArray Parameters() As Variant _

) As String

Parameters

Caption

String that contains the caption to display in the title bar of the menu.

Parameters

Variable number of strings that contain key/value pairs used as options in the menu. The Parameters parameter takes n parameters, where n is an even integer; the parameters contain the following information.

    • Parameter 1: String that contains the key that will be returned if the user selects the first menu option.

    • Parameter 2: String that contains the text displayed for the first menu option.

    • Parameter 3: String that contains the key that will be returned if the user selects the second menu option.

    • Parameter 4: String that contains the text displayed for the second menu option.

    • ...

    • Parameter n-1: String that contains the key that will be returned if the user selects the last menu option.

    • Parameter n: String that contains the text displayed for the last menu option.

Note that the ParamArray keyword in Visual Basic does not indicate an actual array; instead, it indicates a variable number of parameters passed to the function. If an odd number of parameters is passed to this parameter the event handler will throw an error.

Return Values

String that contains the key of the selected menu option. If the cashier cancels the operation the event handler returns an empty string ("").

UI Equivalent Key

None

Example

The following example shows how to use the FireEvent function to display a menu and return the key of the option selected by the user.

JScript

function myShowMenu()

{

  var key = "";

  key = qsBridge.FireEvent("ShowMenu",

    "My Menu",            // Caption for the menu.

    "1", "First Entry",   // Key/value pair for the first option.

    "2",  "Second Entry", // Key/value pair for the second option.

    "3", "Third Entry");  // Key/value pair for the third option.

 

  return key;

}

This function displays the following menu and returns the key of the selected option; for example, if the user selects the "First Entry" option the myShowMenu() function returns "1".
