FOR Element

The FOR element iterates through a collection.

Attributes

Attribute

Value

Description

each

Repeat key

Required. Collection through which to iterate. See the Remarks section for valid values.

Value

None

Child Elements

Any

Remarks

The contents of the FOR element are evaluated for each entry in the specified collection.

The following table shows the valid values for the repeat key.

Repeat key

Description

Entry

Each line-item on a transaction. You must specify this repeat key to use transaction entry variables; see Transaction Entry Template Variables for a list of the transaction entry variables.

KitItem

Each item in a kit.

SalesTaxes

Each defined sales tax.

Tender

Each tender for a transaction.

ReportBatch

Each batch in a report.

ReportTax

Each tax in a report.

ReportTender

Each tender in a report.

ReportDepartment

Each department in a report.

ReportHour

Each hour in a report.

AccountReceivable

Each account receivable entry.

AccountActivity

Each account activity entry.

AccountReceivableActivity

Each activity entry for an account receivable.

AccountReceivableHistory

Each history entry for an account receivable.

PurchaseOrderEntry

Each line-item in a purchase order. You must specify this repeat key to use purchase order variables; see Purchase Order and Inventory Transfer Template Variables for a list of the purchase order variables.

PurchaseOrderEntrySerialEntry

Each serial entry in a purchase order entry. You must specify this repeat key to use purchase order serial entry variables.

Loop

Arbitrary loop.

 
Example

The following example shows how to iterate through the transaction entries to display the description and price of each item in the transaction.

<SET name="PageWidth" custom="true" description="Width of the page" type="vbString">

    2.75

</SET>

 

<TABLE>

    <BORDER>      tbAll        </BORDER>

    <PEN> <TEXT> 20 </TEXT> </PEN>

    <PENTB> <TEXT> 50 </TEXT> </PENTB>

    <PENLR> <TEXT> 100 </TEXT> </PENLR>

    <HEADERSHADE> vbGreen </HEADERSHADE>

 

    <COLUMNHEADER>

        <TEXT>      "Item"  </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.7 </WIDTH>

    </COLUMNHEADER>

    <COLUMNHEADER>

        <TEXT> "Price" </TEXT>

        <ALIGNMENT> "&lt;~" </ALIGNMENT>

        <WIDTH> PageWidth * 0.3 </WIDTH>

    </COLUMNHEADER>

 

    <FOR each="Entry">

        <ROW> Entry.Description "|" Entry.Price </ROW>

    </FOR>

</TABLE>
