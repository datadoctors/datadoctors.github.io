DOCUMENT Element

The DOCUMENT element defines the main body of the receipt template.

Attributes

None

Value

None

Child Elements

Element

Content

Description

NEWPAGE

Optional PAPERCUTPERCENT element that contains a numeric expression between 0 and 100 that indicates the percentage of the receipt to cut when printing is complete

Optional. Inserts a page break.

PICTURE

COLUMNHEADER element

Optional. Displays a picture.

SHIPPINGLABEL

SHIPPINGLABEL element

Optional. Displays a shipping label.

TABLE

TABLE element

Optional. Displays a table.

CALL

CALL element

Optional. Invokes a previously defined subprocedure.

PARAGRAPH

PARAGRAPH element

Optional. Displays a paragraph of text.

BARCODE

BARCODE element

Optional. Displays a barcode.

CIRCLE

CIRCLE element

Optional. Displays a circle.

ELLIPSE

ELLIPSE element

Optional. Displays an ellipse.

LINE

LINE element

Optional. Displays a line.

RECTANGLE

RECTANGLE element

Optional. Displays a rectangle.

MARGIN

MARGIN element

Optional. Redefines the margins.

 

Remarks

The DOCUMENT element is required.

Example

The following example shows how to use the DOCUMENT element.

<SUB name="MySub">

    <PARAGRAPH>

        <TEXT> "Entering MySub." </TEXT>

    </PARAGRAPH>

</SUB>

 

<DOCUMENT>

    <PARAGRAPH>

        <TEXT> "Entering main document." </TEXT>

    </PARAGRAPH>

    <CALL> "MySub" </CALL>

</DOCUMENT>
