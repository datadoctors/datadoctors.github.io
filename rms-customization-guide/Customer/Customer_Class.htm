Customer Class

The Customer class represents a customer.

 

When To Use

The Customer class is used to perform the following tasks.

  • Access customer information, including name, addresses, and user-defined information.

  • Retrieve information about the customer's charge account.

  • View the customer's purchase limit amounts.

 

Procedures

The Customer class defines the following procedures.

Method

Description

ListPurchases

Returns a list of all purchases made by the customer.

LoadShipToAddress

Loads the specified shipping address for the customer.
