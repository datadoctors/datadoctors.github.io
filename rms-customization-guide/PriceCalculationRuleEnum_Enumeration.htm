PriceCalculationRuleEnum Enumeration

The PriceCalculationRuleEnum enumerated type defines the rules used to calculate the price of an item when it is added to a transaction.

Public Enum PriceCalculationRuleEnum

  pricecalculationruleUseMinimum = 0

  pricecalculationruleLevelSupersedesMinimum = 1

  pricecalculationruleUsePriortized = 2

End Enum

Constants

pricecalculationruleUseMinimum

The minimum of the Quantity Discount, Level, Sale, and Buydown prices is used.

pricecalculationruleLevelSupersedesMinimum

The Level price for the item is used. If the Level price is not defined the minimum price is used.

pricecalculationruleUsePriortized

The price used is prioritized as Quantity Discount, Level, Sale, then Buydown. The first defined price is used; for example, if the Sale and Buydown prices are defined but the Quantity Discount and Level prices are not, the Sale price is used because it has priority among the defined prices.

Remarks

The PriceCalculationRuleEnum enumeration is used by the Configuration.PriceCalculationRule property.
