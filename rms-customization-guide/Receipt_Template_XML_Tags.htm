Receipt Template XML Elements

XML elements define the structure of a receipt template. Receipt templates support the following types of XML elements.

  • Conditional Elements: XML elements that determine which elements in the receipt template are rendered.

  • Graphical Elements: XML elements that display information on the receipt.

  • Header Elements: XML elements that contain information used by the entire receipt template.

  • Procedural Elements: XML elements that imitate programmatic functionality.

Receipt templates use the following parent elements.

Element

Description

BARCODE

Optional. Displays a barcode on the receipt.

CALL

Optional. Invokes the contents of a previously defined SUB element.

CIRCLE

Optional. Displays a circle on the receipt.

COLUMNHEADER

Optional. Defines a column in a table.

CURRENCY

Optional. Converts the currency values of all child elements using the format and exchange rate of the specified currency.

DOCUMENT

Required. Defines the main body of the receipt template.

ELLIPSE

Optional. Displays an ellipse on the receipt.

FONT

Required. Defines fonts available in the template.

FOR

Optional. Iterates through a collection.

IF

Optional. Determines which content is rendered on the receipt.

LINE

Optional. Displays a line on the receipt.

MARGIN

Required. Defines the margins for the receipt template.

OPOS

Optional. Defines information for OPOS printers.

PARAGRAPH

Optional. Displays a simple paragraph of text on the receipt.

PICTURE

Optional. Displays a picture on the receipt.

PROPERTIES

Required. Defines information about the XML receipt template.

RECTANGLE

Optional. Displays a rectangle on the receipt.

ROW

Optional. Defines a row in a table.

SET

Optional. Defines a variable for use in the receipt template.

SHIPPINGLABEL

Optional. Displays a shipping label on the receipt.

SUB

Optional. Defines a subprocedure that can be invoked by a CALL element.

TABLE

Optional. Displays a table on the receipt.

TEXT

Optional. Displays text on a receipt.
