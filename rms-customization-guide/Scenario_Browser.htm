Loyalty Add-in Browser

The loyalty program HTML browser is displayed on top of the transaction pane. The loyalty program browser is displayed if a customer is loaded when the loyalty program custom button is clicked. The loyalty pane displayed depends on the status of the loaded customer as follows:

  • Customer not enrolled: Displays the enrollment pane. For more information, see Enrollment Pane in this topic.

  • Customer enrolled: Displays the loyalty status pane. For more information, see Loyalty Status Pane in this topic.

 

Enrollment Pane

The enrollment pane is displayed when the loyalty program custom button is clicked and the loaded customer is not enrolled in the loyalty program. The enrollment pane contains the following elements:

  • Enroll Now button: Click this button to enroll the current customer in the loyalty program.

  • Receive Store Mailings checkbox: Select this checkbox before clicking Enroll Now if the customer wants to receive store mailings as part of their loyalty enrollment.

  • Loyalty level table: Displays the defined loyalty levels.

Loyalty Status Pane

The loyalty status pane is displayed when the loyalty program custom button is clicked and the loaded customer is enrolled in the loyalty program. The loyalty status pane contains the following elements:

  • Loyalty information: Displays the following loyalty program information for the customer:

  • Date on which the customer joined the loyalty program

  • Customer's current loyalty level

  • Customer's current loyalty discount

  • Whether the customer is receiving store mailings

  • Cancel link: Click this link to remove the customer from the loyalty program. All loyalty information for the customer, including current level and purchase amount, is permanently deleted from the database.

  • Yes/No links: The Receive Mailings option that is not selected is displayed as a link. Click the link to change the Receive Mailings option for the customer.
