Transaction.Entries Property

The Entries property gets the details of each entry in the transaction.

Public Property Get Entries( _

) As TransactionEntries

Property Value

TransactionEntries object that contains a collection of TransactionEntry objects, each of which contains information about one entry in the transaction.

Remarks

If you use the SessionClass.Transaction.Entries.Add function to add an item to this collection, the business logic invoked by the POS is bypassed. For example, if you use the Add function to add a serialized item to a transaction, the logic that prompts the cashier to enter a serial number is not invoked, and the serialized item is added to the transaction without a serial number. Also, any options set for the item are ignored, such as "Must enter price at the POS." If your add-in must use this business logic, consider adding the item to the transaction using an HTML script that fires the PerformAddItem event.
