TransactionEntry.LoadSalesRep Function

The LoadSalesRep function sets the sales representative for the transaction entry.

Public Function LoadSalesRep( _

  ByVal ID As Long, _

  ByVal Number As String _

) As StatusCategoryEnum

Parameters

ID

Long value that contains the ID in the database of the sales representative to set for the transaction entry. Set this value to zero to use the Number parameter to identify the sales representative by the identifying string set in the Manager application.

Number

String that contains the identifier of the sales representative to set for the transaction entry. This identifier is set in the Manager application and is not necessarily a number. This parameter is ignored if the ID parameter is not zero.

Return Values

StatusCategoryEnum value that indicates whether the operation was a success.

Remarks

This function sets the sales representative for one transaction entry only. Use the Transaction.LoadSalesRep function to set the sales representative for the entire transaction.
