TransactionEntry.SetItemTax Sub

The SetItemTax subprocedure sets the item tax for the transaction entry.

Public Sub SetItemTax( _

  NewTax As ItemTax, _

  [SetByCashier As Boolean], _

  [UpdateEntryPrice As Boolean] _

)

Parameters

NewTax

ItemTax object to set as the new item tax for the transaction entry.

SetByCashier

Optional. Boolean value that specifies whether the item tax was changed by the cashier. This parameter is used for tracking purposes. The default value is TRUE.

UpdateEntryPrice

Optional. Boolean value used by VAT tax systems that specifies whether the prices in the TransactionItem object referenced by the Item property should be updated. The default value is TRUE.

Remarks

This subprocedure sets the item tax for the transaction entry as well as the item taxes for all other entries in the transaction that contain the same item.

The Transaction.ItemTaxes property contains a list of all defined item taxes.
