TransactionEntry Class

The TransactionEntry class represents a single line entry in the transaction.

 

When To Use

The TransactionEntry class is used to perform the following tasks.

  • View and set the prices, discounts, quantities, and sales tax for the entry.

  • Load or clear the sales representative for the entry.

  • Set or view the serial numbers for the item in the entry.

  • Set or view the expiration date for a voucher.

 

Procedures

The TransactionEntry class defines the following procedures.

Method

Description

ClearSalesRep

Clears the sales representative for the transaction entry.

LoadSalesRep

Sets the sales representative for the transaction entry.

LoadSerial

Sets the serial numbers for the serialized item in the transaction entry.

RevertToSavedPrice

Restores the price that was saved with the SavePrice subprocedure.

SavePrice

Temporarily saves the price for the transaction entry.

SetDiscount

Calculates the price for the transaction entry based on the specified discount method and percent.

SetExtendedPrice

Sets the price for all quantities of the item in the transaction entry.

SetItemTax

Sets the item tax for the transaction entry.

SetPrice

Sets the price for a single item in the transaction entry.

SetQuantityOnOrder

Sets the quantity of ordered items in the transaction entry.

SetQuantityPurchased

Sets the quantity of purchased items for the transaction entry.

SetTaxable

Sets the taxable status of the transaction entry.
