Store Information Template Variables

The following receipt template variables contain information about the store.

Variable

Description

Configuration.StoreName

Name of the store.

Configuration.StoreAddress

First line of the address of the store.

Configuration.StoreAddress2

Second line of the address of the store.

Configuration.StoreCity

City of the address of the store.

Configuration.StoreState

State code of the address of the store.

Configuration.StoreZip

Postal code of the address of the store.

Configuration.StorePhone

Telephone number of the store.

Configuration.StoreFax

Facsimile number of the store.

Configuration.CityStateZip

City, state code, and postal code of the address of the store.

Configuration.ZipCity

Postal code and city of the address of the store.

Configuration.LogoFilename

File name of the logo image.
