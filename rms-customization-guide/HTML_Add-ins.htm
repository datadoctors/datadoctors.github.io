HTML Add-ins

HTML add-ins extend the Store Operations POS functionality by supporting customized retailer features and requirements. The POS contains three embedded HTML browsers; HTML add-ins can use these browsers to access Store Operations data and display transaction information. The three browsers are the following:

  • The HTML Status Bar displays Web pages and information about the current transaction at the top of the POS screen.

  • The HTML Browser replaces the transaction grid on the POS screen when launched from a custom POS button. Users can interact with this browser to perform complex tasks.

  • The Net Display runs on a secondary customer-facing monitor and displays two HTML panes: a receipt pane and an advertising pane. The receipt pane shows the current transaction. The advertising pane displays merchant-selected ads or Web pages.

Note: There is a fourth HTML browser, referred to as the Internal HTML window. This browser launches an external website and does not interact with QSRules or the user interface workflow.

 

HTML Status Bar

Use the HTML status bar at the top of the POS window to display status information about the current transaction. This HTML pane can also display a non-interactive Web page. The HTML status bar can only be used to display information; users cannot interact with the status bar because the transaction grid always has the focus.

Store Operations includes two HTML script files that you can use for the status bar. Status.HTM displays the customer status; LineItem.HTM displays information about the current transaction entry. These files are located in the HTML folder under the folder in which Store Operations is installed; the default location is C:\Program Files\Microsoft Retail Management System\Store Operations\HTML.

Net Display

Use the HTML panes on the Net Display to display HTML pages to the customer. The Net Display runs on a secondary customer-facing monitor and consists of two panes.

  • Receipt Pane – Displays a graphical receipt to the customer. The Receipt.HTM script included with Store Operations displays a receipt in this pane as shown in the image below.

  • Advertising Pane – Displays advertisements to the customer. Because the Net Display window can access Store Operations business logic, the advertisements can be modified to display information such as the current customer and items on the transaction.
