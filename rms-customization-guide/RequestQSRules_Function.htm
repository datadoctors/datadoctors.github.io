QSBridge.RequestQSRules Function

The RequestQSRules function returns a reference to the current SessionClass object within QSRules. This reference can then be used to access all of the QSRules objects.

Public Function RequestQSRules( _

) As Object

Return Values

Reference to the head SessionClass object.

Example

The following example uses the QSBridge.RequestQSRules function to retrieve the shipping charge amount from the QSRules Shipping object.

VBScript

<SCRIPT language="vbscript">

<!--

Dim QSRules

Dim ShippingCharge

Set QSRules = qsBridge.RequestQSRules()

If Not QSRules Is Nothing Then

   ShippingCharge = QSRules.Transaction.Shipping.Charge

End If

//-->

</SCRIPT>

JScript

<SCRIPT language="jscript">

<!--

var qsRules = qsBridge.RequestQSRules();

var shippingCharge = 0;

if (qsRules != null)

   shippingCharge = qsRules.Transaction.Shipping.Charge;

qsBridge.FireEvent("RefreshDisplay");

//-->

</SCRIPT>
