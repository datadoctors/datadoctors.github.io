CURRENCY Element

The CURRENCY element converts the currency values of all child elements using the format and exchange rate of the specified currency.

Attributes

Attribute

Value

Description

code

String variable name

Required. Code of the currency to use within the element. This attribute must refer to a string variable whose value is a valid currency code in the current RMS database.

Value

None

Child Elements

Any

Example

The following example displays the subtotal for the transaction using the format and exchange rate of the entry in the Currency table with the Code value "EUR".

<SET name="CurrencyCode" custom="true" description="Secondary Currency Code" type="vbstring"> "EUR" </SET>

 

<CURRENCY code="CurrencyCode">

    <ROW> Transaction.SubTotal </ROW>

</CURRENCY>
