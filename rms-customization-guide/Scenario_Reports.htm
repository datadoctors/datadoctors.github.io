Loyalty Add-in Reports

The loyalty add-in generates reports to track vital information about the loyalty program. The add-in generates the following reports:

  • Loyalty Effectiveness report: The Loyalty Effectiveness report compares the purchases each customer made before joining the loyalty program to the purchases made after joining the program. For more information, see Loyalty Effectiveness Report in this topic.

  • Cashier Sign-up report: The Cashier Sign-up Report shows the effectiveness of each cashier in signing up customers for the loyalty program. For more information, see Cashier Sign-up Report in this topic.

  • Customer Birthday report: The Customer Birthday report lists the customers enrolled in the loyalty program whose birthday falls within a specified date range. For more information, see Customer Birthday Report in this topic.

 

Loyalty Effectiveness Report

The Loyalty Effectiveness report compares the purchases each customer made before joining the loyalty program to the purchases made after joining the program. This report contains the following fields:

  • Customer name

  • Date on which the customer account was created

  • Date on which the customer joined the loyalty program

  • Customer's loyalty level

  • Total purchases made by the customer before joining the loyalty program (excluding sales tax)

  • Total purchases made by the customer after joining the loyalty program (excluding sales tax)

  • Average monthly purchases made by the customer before joining the loyalty program (excluding sales tax)

  • Average monthly purchases made by the customer after joining the loyalty program (excluding sales tax)

The Loyalty Effectiveness report can be filtered by the following fields:

  • Customer name

  • Date on which the customer account was created

  • Date on which the customer joined the loyalty program

  • Customer's loyalty level

 

Cashier Sign-up Report

The Cashier Sign-up report shows the effectiveness of each cashier in signing up customers for the loyalty program. This report contains the following fields:

  • Cashier name

  • Number of customers served by the cashier that are not enrolled in the loyalty program

  • Number of customers signed up for the loyalty program by the cashier

  • Percentage of customers served by the cashier signed up for the loyalty program by the cashier

The Cashier Sign-up report can be filtered by the following fields:

  • Cashier name

  • Time period for which to generate the report

 

Customer Birthday Report

The Customer Birthday report lists the customers enrolled in the loyalty program whose birthday falls within a specified date range. This information can be used to send birthday coupons to loyalty customers. This report contains the following fields:

  • Customer name

  • Customer's date of birth

  • Date on which the customer joined the loyalty program

  • Customer's loyalty level

  • Total purchases made by the customer after joining the loyalty program (excluding sales tax)

The Customer Birthday report can be filtered by the following fields:

  • Customer's date of birth

  • Customer's loyalty level
